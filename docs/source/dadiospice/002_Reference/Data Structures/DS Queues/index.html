队列结构（Queue）

一个队列结构是一个先进先出（FIFO，First-In-First-Out）的数据结构。

队列结构是有点类似于堆栈结构，但是它的基本原则是先进先出（FIFO）。首先放入队列中的值也会是首先被移除的值。你可以想象这就是在商场排队交款一样，在那里队伍中靠前的人会首先交款并离开商店。队列通常被用于存储一个实例即将执行的一系列动作或行为（比如说以脚本的格式）但事实上还有其他用途。

注意： 和所有动态资源一样，数据结构会占据大量内存，并且因此应该在不再被使用的时候总是被销毁以防止内存泄漏将拖慢你的游戏并最终使你的游戏崩溃。

下列函数用于处理队列结构：

  1. ds_queue_create
  2. ds_queue_destroy
  3. ds_queue_clear
  4. ds_queue_empty
  5. ds_queue_size
  6. ds_queue_dequeue
  7. ds_queue_enqueue
  8. ds_queue_head
  9. ds_queue_tail
  10. ds_queue_copy
  11. ds_queue_read
  12. ds_queue_write
注意： 如果你需要检测一个数据结构是否存在，你可以使用函数ds_exists()

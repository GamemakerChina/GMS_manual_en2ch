ds_list_clear

清除来自一个被给定列表的所有数据。

语法:

ds_list_clear(id);


参数 描述
id 待清除的数据结构id。


返回值: 无


描述

使用这个函数你能够清除来自于被给定列表数据结构中的全部数据。这不会销毁这个数据结构（对于那种你应当使用ds_list_destroy）它仅仅从中擦除所有数据并且返还一个空列表。


举例:

if count = 15 && !ds_list_empty(command_list)
   {
   ds_list_clear(command_list);
   alarm[0] = room_speed;
   ai_count = 0;
   }

上述代码检测一个变量来观察它是否已经到达一个特定的值，如果是的它就清空被索引为变量"command_list"的列表结构，设定一个闹钟，并且将这个变量重置为0.

exit 语句

这里将解释怎样使用“exit”语句。

“exit”语句的结构：

exit;


“Exit”简单的结束当前脚本或事件的运行。注意这儿的不同点，如果只是简单的在脚本中使用exit，它将跳出脚本并返回调用脚本的代码的位置，然后如果在物体的事件中使用，它将跳出 整个事件，即使后面还有独立的代码块。通常它是用来避免运行一个特定的代码块，比如碰撞事件。下面的代码给出了一个简单的例子：

{
if invisible exit;
while (place_meeting(x, y))
   {
   x -= lengthdir_x(1, direction - 180);
   y -= lengthdir_y(1, direction - 180);
   }
}


上述代码检测一个变量，如果为真它将退出代码块，否则继续运行后面的代码。

注意： 它不会结束游戏的运行。如果，你需要使用game_end()。

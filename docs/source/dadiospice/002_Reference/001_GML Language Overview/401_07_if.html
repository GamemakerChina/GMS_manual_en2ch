if 语句

这节概述了如何使用"if"语句。

一个简单的"if"语句具有这种形式：

if (<表达式>) <语句>


或者它也可以有略微复杂的"if... else..."形式：

if (<表达式>) <语句1> else <语句2>


在这个情况下，对这个表达式（expression）进行求值，并且如果（近似）值是<=0（false），语句后面的else被执行，否则（true）另一个语句被执行。经常在if后面的语句放上花括号是一个好习惯，并且为每一个语句另起一行，所以最终代码将会有这种格式：

if (<表达式>)
   {
   <语句>
   ...
   <语句>
   }
else
   {
   <语句>
   }


作为一个小例子，想一想下面的代码，哪一块将移动实例朝着x=200移动：

if (x < 200)
   {
   x += 4;
   }
else
   {
   x -= 4;
   }

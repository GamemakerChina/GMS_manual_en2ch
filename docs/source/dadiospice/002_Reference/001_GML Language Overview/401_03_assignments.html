赋值

本篇文章是与变量赋值相关的内容。

赋值语句把一个值赋予一个变量，其格式如下：

<变量> = <表达式>;


其中，表达式（expression）可以是一个值，也可以是复杂的代数式，除了将一个值赋给一个变量，还可以使用 “+=”在变量存储的值上加一个值，比如：

a += b;


类似地，也可以用 -=实现自减，使用 *=自乘，使用 /=自除，或者也可以使用位运算符 |=、 &=和 ^=。你甚至可以用以下符号加减一 ： ++，--。

physics_test_overlap

检验两个支架在一个被给定的位置是否有交叠。

语法：

physics_test_overlap(xpos, ypos, angle, obj);


参数 描述
xpos 房间内待检测的x位置
ypos 房间内待检测的y位置
angle 待检测的角度（调用此函数实例的）
obj 待检测的物体


返回： 布尔型


描述

这个函数可以被用于检验和观察一个物理实体（例如：实例的支架）是否存在交叠，或者将会发生交叠当旋转到这个角度并发生于房间内被给定的位置。"angle"参数是在待检测位置调用此函数的实例所拥有的（或将会有的）角度，并且"obj"参数可以是单个实例的id，也可以是物体索引或者关键词常量all or other.


举例：

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
   {
   alarm[0] = room_speed;
   ignited = true;
   }

上述代码将会为物理支架的交叠而检测一个位置，并且如果有这个位置，则它就设定一个变量和一个计时器。

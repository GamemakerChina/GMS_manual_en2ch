show_message

显示一个弹出消息。

语法:

show_message(str);


参数 描述
str 待显示在弹出消息中的字符串。


返回值: 无


描述

这个函数创建一个弹出消息框，其显示在代码中被定义的字符串和一个标有"Ok"的按钮来关闭。

注意: 这个函数仅供调试！


举例:

var i, tot;
tot = 0;
for (i = 0; i < 10; i += 1)
   {
   tot += inv[i];
   }
show_message("Total = " + string(tot));

上述代码将会迭代被存储在数组"inv"中的值并且将其加到变量"tot"中之后显示一个带有总数的消息。

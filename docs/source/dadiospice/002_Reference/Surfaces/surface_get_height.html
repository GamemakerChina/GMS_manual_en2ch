surface_get_height

查询表面层的像素高度。

语法：

surface_get_height(surface_id);


参数 描述
surface_id 将要获取高度的表面层ID。


返回： 实数


描述

这个函数简单的返回索引表面的像素高度。

注： 由于储存在显存里，表面层运行时存在随时停止并退出的可能性。在直接引用它们之前，你需要 一直 使用 surface_exists 检测表面层是否退出。更多信息请查阅 表面层。


举例：

sh = surface_get_height(surf);

上述代码将在变量sh中储存索引变量为surf的表面图层的高度。

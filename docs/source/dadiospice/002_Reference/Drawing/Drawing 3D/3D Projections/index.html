3D 投影

本章节详解 3D 投影相关的函数

3D游戏中，我们需要改变观察世间的方式。比如FPS类游戏，你需要摄像机站在 X-Y 平面的上方，视线沿着这个X-Y平面，望向远方。 为了能够正确设置投影，你应该在房间的第一个对象中进行配置（你能在 Room Editor-Setting-Instance Order 窗口中设置）。尽早进行相关的设置，否则会造成一些异常情 况。 注意： When GameMaker starts drawing a room it will set the view point back to the default position, so the first thing you must do when drawing the scene is set is to the projection you want. This must be done in the draw event of an instance!

你可以使用以下这些函数，变更观察3D世界的方式。

  1. d3d_set_projection
  2. d3d_set_projection_ext
  3. d3d_set_projection_ortho
  4. d3d_set_projection_perspective

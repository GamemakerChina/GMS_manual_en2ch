random_set_seed

设定用于从中取得随机数的随机数种子。

语法:

random_set_seed(val);


参数 描述
val 待设定的随机数种子


返回值: 无


描述

为了产生一个随机数GameMaker: Studio用一个随机数种子数开始。使用这个函数你能够将这个随机数种子设定为一个已知值并以此来“强迫”所有随机事件产出的结果在后续程序运行的时候是相同的。举个例子，这个函数可以和函数random_get_seed连用来创造一个程序性生成的内容并且无需庞大的游戏存档地保存这个结果。（你只需要保存种子，并不需要别的东西）。你可能应该需要真正意义上的处处随机结果，则你应该使用函数randomize()

注意: 尽管这个种子在每个同一平台上会给出已一致的结果，结果可能在在平台之间会有所变化因为不同平台工作模式的不同。


举例:

if debug random_set_seed(1);

上述代码只有在变量"debug"为真的时候设定随机数种子为1.

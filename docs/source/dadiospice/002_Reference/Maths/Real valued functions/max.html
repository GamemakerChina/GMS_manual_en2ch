max

返回最多16个数中的最大值。

语法:

max(val1, val2, ... val15);


参数 描述
val0 ... val15 待比较的值。


返回值: 实数型


描述

这个函数返回输入值中最大的，其中它最大可以有16个值 举个例子max(12, 96, 32, 75)将返回96因为是所有输入值中最大的。


举例:

x = max(x, 0);

这将会组织玩家从左边离开房间。这是通过恒定地设定它的x为自身和为0中较大的。如果玩家从左边离开了， 它的x将小于0（也就是：负数）。所以它将直接设定回来。

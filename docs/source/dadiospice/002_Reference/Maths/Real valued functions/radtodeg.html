radtodeg

将弧度制转换为角度制。

语法:

radtodeg(rad);


参数 描述
rad 带转换的弧度。


返回值: 实数型


描述

一旦你使用sin或者cos等完成了你的运算，最终结果是弧度制的。 这可能不总是你想要的并且因此欲将弧度转换为角度我们使用这个函数。 举个例子radtodeg(sin(180)) 将会返回-45度。这个函数利用下面的公式将弧度转换为角度：

角度 = 弧度 * 180 / π;


举例:

val = radtodeg( pi );

这将把val设为180。

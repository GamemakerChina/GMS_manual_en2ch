log10

返回n以10为底的对数.（即“常用对数”）

语法:

log10(n);


参数 描述
n 输入值。


返回值: 实数型


描述

这个函数和函数 log2(n)很相似, 只不过 log10(n)获取我们为了得到n需要把若干10相乘在一起的个数。所以 log10(100) 返回 2, 因为10乘以自身就会得到100， log10(1000) 返回 3 因为 10×10×10=1000 等等... 这个数 不必是能被10整除的数，然而，这也仍然会返回一个近似值， 举个例子 如果我们有log10(101) 我们会得到2.00432137... 就是101能除以10的次数。⑩


⑩：简单的说，n的常用对数就是n是10的多少次幂，即10的多少次方是n。


举例:

logval = log10(num);

上述代码会得到被存储进变量"num"的常用对数 。

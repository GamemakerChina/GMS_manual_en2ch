log2

返回以2为底n的对数。（二进制对数）①


①：此处删去的内容被证实与后文矛盾，而后文内容显然是合理的。

语法:

log2(n);


参数 描述
n 输入值.


返回值: 实数型


描述

从根本上说这个函数求“你需要多少个2乘在一起能得到n？”。 在编程过程中使用这个数的一个普遍例证是计算表示一个数需要多少位②。 欲做这个我们就需要计算这个数以2为底的对数 ， 向下取整再+1 - 举个例子 log2(100) 返回 6.643856, 向下取整是6, 对它加一然后它就能给我们7。③ 所以表示100我们需要7个位。这也能用于描述指数级增长或衰退的程度，且也常被称为 二进制对数④.

②：此处的位为计算机概念，即bit，1B = 8bit。
③：汗= =……为什么不直接向上取整呢？
④：简单地说，二进制对数就是n是2的多少次幂，即n对于2的幂指数。


举例:

colourbits = floor(log2(colour)) - 1;

上述代码将代表这个数所需要的位数存入变量"colour"中。

cloud_file_save 云端文件保存

保存文件（同字符串）到所选的云服务器中。

语法：

cloud_file_save(file, description);


参数 描述
string 将被上传的文件(同i符串)。
description 所存储数据的简要说明。


返回： 实数


描述

这个函数将提交一个文件到所选云服务用于存储。该函数将返回惟一的id 值，那么应该使用适当的异步事件识别 ds_map 返回作为云服务的“回调（call back）”。该文件应该包含你所需要保存的游戏的 全部 信息，你只能上传一个二进制数据（ "data blob" ）到服务器上, 并且再次运行这个函数将覆盖之前存储的值 (如同使用 cloud_string_save 一样). 描述文字应该尽量的短，用于描述保存操作, 例如： "Level2, Stage2".

想要进一步了解异步函数返回的数据，请查阅函数 cloud_synchronise.


举例：

var t_str = "";
for (var i = 0; i < 10; i++;)
   {
   t_str += string(global.Highscore[i]) + "|"
   }
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);
save_check = cloud_file_save("Highscores.txt", "Current Highscores");

以上的代码创建了一个字符串的值存储在全局数组“Highscores”中,然后将该字符串写入本地存储。该文件之后被提交到云端服务器中存储。

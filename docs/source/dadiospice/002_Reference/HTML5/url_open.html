url_open

通过浏览器访问指定链接

语法：

url_open(url);


参数 说明
url 网站链接


返回： N/A


说明

在对应的设备上，程序自动打开浏览器并访问指定的网址。如果是html5平台，在本页面跳转指定的网址

示例：

url_open( 'http://yoyogames.com' );

调用函数后，程序将打开浏览器，并访问 yoyogames 的官方首页

view_vspeed

视野的最大垂直速度。

语法：

view_vspeed[0...7]


返回： 实数


描述

视野有跟随物体时（这个可以用 view_object 变量设置或者在 房间编辑器中设置），这个变量控制控制视野的垂直速度。谨慎设置此值，因为设置过低将导致实例比视野移动更快，然后消失在视野外面，直至停下后才再次跟上。


举例：

view_hspeed[0] = view_object[0].hspeed;
view_vspeed[0] = view_object[0].vspeed;

上述代码设置视野的移动速度与跟随的物体的速度一致。

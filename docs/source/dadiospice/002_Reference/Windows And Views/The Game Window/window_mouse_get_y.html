window_mouse_get_y

返回窗口内的鼠标y坐标。

语法：

window_mouse_get_y();


返回： 实数


描述

有人这个函数，你可获得游戏窗口的鼠标指针的y坐标（像素为单位），如果是HTML5游戏则位于浏览器之内，如果是 Windows、 Ubuntu (Linux)或 MacOS 游戏，则位于显示屏以内。.


举例：

wy = window_mouse_get_y();

上述代码储存当前窗口内的鼠标y轴坐标至变量“wy”。


返回: 游戏窗口

window_handle

返回游戏窗口的ID。

语法：

window_handle();


返回： 指针


描述

有人这个函数，你可以获取内置的窗口ID的值 (a 指针)。当他们需要使用window handle 在DLL代码中调用 Win32 API的时候，这个函数对于插件作者非常有用。

注意: 这个函数只用于 Windows 和 Mac OSX (Mac OSX 包含了一个指向 NSWindow 类的指针)。


举例：

win_id = windows_handle();

上述代码将在变量中保存游戏窗口的id。


返回: 游戏窗口

current_time

返回系统启动到现在的毫秒数

语法:

current_time;


返回: Real


描述

这个 只读 变量将返回系统启动以来经过的毫秒数


例子:

if current_time > 600000
   {
   msg = show_question_async("Would you like to rate?");
   }

上面的代码检查系统启动是否超过10分钟，超过则询问用户一个问题


返回: 时间与日期

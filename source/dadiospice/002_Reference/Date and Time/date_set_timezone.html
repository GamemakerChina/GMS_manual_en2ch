date_set_timezone

设置时区用于所有的日期和时间函数

语法:

date_set_timezone(timezone);


内容提要
 描述
时区 时区（Time Zone)是地球上的区域使用同一个时间定义。


返回: N/A


描述

设置时区来影响其他时间函数，这个时区可以是本地(如系统设定的)或UTC,你可以使用以下常量定义使用(在默认情况下这是本地时间):

常数 描述
timezone_local 使用系统本地的时区设置
timezone_utc 使用协调世界时


例子:

if date_get_timezone() != timezone_utc
   {
   date_set_timezone(timezone_utc);
   }

这段代码检查游戏设置的基本时区,如果不是UTC,然后更改。


返回: 日期与时间

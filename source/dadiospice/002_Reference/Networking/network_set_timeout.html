network_set_timeout

设置客户端的socket，在发送请求和获得响应时的最大时间（超时）

语法：

network_set_timeout(socket, read_timeout, write_timeout);


参数 说明
socket The id of the socket to use.
read_timeout 从服务端下载数据，可容忍的最大时间（毫秒），超时会失败且不会触发任何事件。
write_timeout 向服务端上传数据，可容忍的最大时间（毫秒），超时会失败且不会触发任何事件。


返回值 N/A


说明

这个函数用于设置传输数据的超时，如果超过时间还没有结果，那么传输会被终止，同时不会有任何返回值以及事件被触发，所以（比如）你需要自己设置一个alarm 来处理。如果你想设置socket连接超时，可以使用network_set_config。


示例：

network_set_timeout(sock, 3000, 3000);

以上代码设置了变量 sock 所存储的 socket id 的传输超时为3秒钟。

network_set_config

设置联网相关的配置

语法：

network_set_config(config_value, setting);


参数 说明
config_value 配置项（可用参数在下文列表中）
setting 配置项的值


返回值：字符串


说明

这个函数设置联网相关的配置，可配置项是 常数，列于下方列表:

可配置项 说明 可配置项的值
network_config_connect_timeout 设置连接超时 一个整数，单位毫秒
network_config_use_non_blocking_socket 设置 GameMaker: Studio 在连接时是否堵塞程序 0 = 阻塞, 1 = 不阻塞



可选1 (network_config_connect_timeout) 不改变连接状态，除非连接超时。建议不要设置过长或过短，最好局域网1000，网络游戏4000。

可选2 (network_config_use_non_blocking_socket) Socket在连接后触发 异步回调事件)，告知程序连接结果。在没有结果前，你不能使用这个 socket id 发送或接收数据。


示例：

network_set_config(network_config_connect_timeout, 1000);
network_set_config(network_config_use_non_blocking_socket, 1);

以上代码，配置连接超时1000ms，并且连接时不堵塞程序。


返回： 联网

云服务

这些函数可以用来将数据同步到不同的服务器上。

在游戏开发过程中，常常需要将游戏的状态信息保存在特定类型的文件中，但是把这些信息保存在本地设备上在大多数时候并不是一个好的选择，因为当玩家在卸载游戏之后重新安装时，这些数据可能会丢失。为了防止这种情况的出现您可以使用各种不同的云服务 ，用来提供可以修改和检索数据的云端据存储服务。需要注意的是这个函数在每个游戏中被限制使用 一个 单一的"二进制"数据, 每当你发送一个新的保存请求到云服务器时，要确认它是一个字符串还是一个文件，因为它将覆盖你先前提交的任何已存储的数据。

GameMaker: Studio 提供这几个简单的函数来实现 异步云事件(Asynchronous Cloud Event)。下面的网页将详细介绍这些函数：

  1. cloud_synchronise 云端同步
  2. cloud_string_save 云端字符串保存
  3. cloud_file_save 云端文件保存

注意：目前仅支持亚马逊 WhisperSync的服务，你需要在全局设置（Global Game Settings）中的 Android Tab 来激活这个功能。

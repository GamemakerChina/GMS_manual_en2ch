物体与实例

这节有所有直接与物体和实例相关的功能

为了在GameMaker: Studio中制作一个游戏，你将需要首先定义物体并且将其添加到资源树中。一个物体可以包含许多不同的事件并且在每个事件中你既可以使用GML代码也可以使用D'n'D按钮来为那个物体定义行为。然而，这个物体并没有被放置到游戏中，而是表现为一个被放进房间的实例的模板。一个实例从根本上说是被放在你游戏的房间中的一个物体复制品，并且因此，以这种方式，你将会创建一个墙体物体并接下来放置100个它的实例于房间中来创建你的游戏世界。

另一件需要弄清楚的关于实例和物体的事情就是一旦一个实例被创建，它能够通过D'n'D按钮或代码以个体作为基础令这些改变都不必影响相同物体的其余实例而被调配并改变。比如说，你能够使用房间编辑器右键实例菜单来改变一个单独的实例来具有一个不同的颜色或方向并且这些改变将会只作用于当前实例。 All further instances will be created exactly the same as the defined object template.

instance_id

返回房间内一个实例的id标识。

语法:

instance_id[0 ... n];


返回值: 实数


描述

这个只读数组承载了房间内每个活跃实例的所有id标志。这也就意味着如果你使用任意一个instance_deactivate函数，那些被冻结的实例将不会被包括在这个数组中（如果你曾经之前使用过一个来自于这个数组的值，它现在就会返回关键字noone）。


举例:

var i;
for (i = 0; i < instance_count; i += 1;)
   {
   with (instance_id[i]) speed += 0.1;
   }

上述代码将会迭代房间内所有实例并且将它们的速度增加0.1.

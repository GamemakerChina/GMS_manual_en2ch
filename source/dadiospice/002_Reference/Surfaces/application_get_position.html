application_get_position

获取应用表面层默认绘制的位置

语法：

application_get_position();


返回： 数组


描述

当你在全局游戏设定中为一个目标平台打开“保持纵横比”，GameMaker: Studio 将自动设置应用表面层的绘制位置，让显示器正确的居中放大显示。但是如果你想自己操作表面进行绘制，这个函数可以让你 在显示器或窗口方便的找到表面层绘制的位置，然后在那里绘制，或者对齐GUI图片或者上面绘制图片。

这个函数将返回包含四个键值的数组，键0和键1是表面左上角x/y的位置，键2和键3是表面右下x/y的位置，所有的这些值响应显示器及窗口发生变化。


举例：

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

上述代码获取应用表面层的位置，以及相对于显示器或窗口的绝对长宽值，保存到四个变量上备用。


返回: 表面层

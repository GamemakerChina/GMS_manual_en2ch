surface_create

创建一个表面。

用法:

surface_create(w, h);

 

参数 描述
w 被创建的表面的宽度。
h 创建的表面的高度。

 

返回:实数

 

描述

这个函数被用来创建一个表面图层，而且将会返回这个表面的索引地址，这个索引应该储存在一个变量上，将来可以调用它。第一次创建的表面图层可能会包含一些“噪音”，基本上是因为它只是为了特别目的而放置一边的内存区域（并且那个内存可能仍然包含信息），所以你可能会需要在使用之前用draw_clear_alpha这样的函数清理这个表面图层。

强烈建议所有创建的表面图层大小设置为2的N次方，比如16、128、512、1024像素。不是所有的平台都有必要这样做（比如Windows和MacOS），但是这样做肯定也会增加那些平台的兼容性。然而，对于HTML5和其它设备，记住这非常重要，或许你以后就会出现问题。

 

例子:

if !surface_exists(surf)
   {
   surf = surface_create(1024, 1024);
   surface_set_target(surf);
   draw_clear_alpha(c_black, 0);
   surface_reset_target();
   view_surface_id[0] = surf;
   }

上述代码将检测是否存在一个表面图层，如果没有则创建一个1024×1024px大小的表面图层，并指定给索引为变量“surf”。然后绘制目标设置给这个新的表面图层，接下来是清除表面，设置为透明，再是将绘制目标设置给显示器。最后，给这个表面图层指定给一个视野。

surface_copy_part

复制源表面图层的一部分到目标表面图层。 (没有任何形式的混合。)

语法：

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);


参数 描述
destination 将要复制到的其它的表面层的索引ID。
x 复制到的x坐标。
y 复制到的y坐标。
source 被复制的表面层ID。
xs 从源表面层复制的x位置。
ys 从源表面层复制的y位置
ws 从源表面层复制的区域的宽度。
hs 从源表面层复制的区域的高度。


返回： N/A


描述

这个函数简单的让图像从一个表面图层复制到另外一个表面特殊的位{0/}置（(0,0)表示在表面图层的左上角）。你可以指定源表面复制区域的本地x和y坐标，以及选区的宽度和高度。请注意，坐标是基于 表面层的大小，而不是表面层在房间绘制的位置（查看 Surfaces目录页获得更多的信息）。如果目标表面已经有内容了，将会被拷贝的表面层重写，这个函数 不会用任何方式改变源表面图层。

注意 : 由于储存在显存里，表面层运行时存在随时停止并退出的可能性。在直接引用他们之前， 你需要 一直使用 surface_exists检测表面层是否退出。更多信息请查阅 表面层.


例如：

if view_current == 0
{
surface_copy_part(surf, 0, 0, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y, temp_surf);
}
else
{
draw_surface(surf, 0, 0);
}

上述代码将检测当前被绘制的视野是否为view[0]，如果是，则复制变量为"temp_surf"的表面到变量为 "surf"的表面。被复制的区域相当于一个矩形，截取自表面内相对鼠标的位置，它将在view[1]里绘制。如果当前视野不是view[0]，则绘制 "surf"这个表面到屏幕上。

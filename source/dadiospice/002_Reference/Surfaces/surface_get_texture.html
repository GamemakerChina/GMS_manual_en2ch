surface_get_texture

返回给定表面层的纹理ID。

语法：

surface_get_texture(surface_id);


参数 描述
surface_id 即将获取纹理的表面层ID。


返回： 指针


描述

这个函数返回表面层纹理的特殊 指针 。这个值可用于其它的绘制函数，特别是普通的3D和2D图元函数。

注: 由于储存在显存里，表面层运行时存在随时停止并退出的可能性。在直接引用它们之前，你需要 一直 使用 surface_exists 检测表面层是否退出。更多信息请查阅 表面层.


举例：

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglefan, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

上述代码将绘制四个顶点的三角形面，纹理采集自之前创建好的表面。


返回: 表面层

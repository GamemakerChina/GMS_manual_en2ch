get_integer

显示一个弹出信息以要求用户输入一个整数

语法:

get_integer(str, def);


参数 描述
str 弹出信息中显示的字符串
def 文本框中的默认值


返回值: 实数①
①：根据后文内容可知，实际上应当是整数。


描述

这创建了一个弹出窗口显示一个自定义消息，带有一个标签为"Ok"的按钮，以及用户输入一个整数值的提示。这个函数将会以整数形式返回被输入的内容，或者如果没有东西被输入的默认值。



注意: 这个函数只用于调试用途。若你想在你最终游戏中要求这个功能，请使用get_integer_async.


举例:

global.level = get_integer("Level to test?", 1);

上述代码将会显示一个信息要求用户选择关卡用于测试。这个返回值将被存入全局变量"global.level"中。

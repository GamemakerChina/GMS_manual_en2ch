fps_real

返回实际的帧速率（帧每秒）。

语法:

fps_real


返回值: 实数①
①：根据后文内容可知，实际上应当是整数。


描述

在GameMaker: Studio中有两个主要的方式能够被用来识别你的游戏在以什么样的速度运行着。room_speed（和在房间编辑器里所指定的一样）和fps（帧每秒）。这些值经常被混淆，但是从根本上说一个是GameMaker: Studio每秒应当完成的步数，而另一个（fps）是GameMaker: Studio每秒实际完成的CPU步数，而这个值通常比房间速度高德多，但是会随着你的游戏变得越来越复杂并且使用更多的处理能力来位置房间速度而下降。

这个只读变量将当前帧速率返回为一个整数值。请注意这个数只会在游戏的每步更新一次并且可能会从一个值“跳”到另一个，但这是相当正常的。


举例:

if debug_mode
   {
   draw_text(32, 32, "FPS = " + string(fps_real));
   }

上述代码将会检查游戏是否以调试模式运行，并且如果是则它会在屏幕上显示当前真正的帧速率。

code_is_compiled

检查代码是否呗使用YoYo Compiler编译。

语法:

code_is_compiled();


返回值: 布尔


描述

如果这个平台在虚拟机外部编译了代码，这个函数就会返回true，比如对于YYC和JS平台。


举例:

if code_is_compiled()
   {
   show_debug_message("Compiler okay!");
   }

上述代码检测游戏是否被使用YoYo Compiler编译并且如果是则显示一个调试信息。

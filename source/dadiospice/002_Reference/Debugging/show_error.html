show_error

显示一个带有自定义错误字符串的弹出消息。

语法:

show_error(str, abort);


参数 描述
str 弹出消息中显示的字符串。
abort 是(true)否(false)中止


返回值: 无


描述

这个函数将会显示一个自定义字符串作为一个错误信息并且如果必要则会给出选项来中止后续游戏。

注意: 这个函数仅供调试！


举例:

if room != rm_Dungeon
{
show_error("Error: Went to wrong area. Aborting game.", true);
}

上述代码将会检查玩家所在房间是否为"rm_Dungeon"，如果不是，则它会显示一个错误消息并且强制用户中止游戏。

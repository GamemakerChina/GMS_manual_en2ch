show_debug_message

在编译窗口中显示一个调试信息。

语法:

show_debug_message(string);


参数 描述
string 待显示的自定义信息。


返回值: 无


描述

这个函数将会创建一个在运行期间被显示在编译窗口的自定义调试信息.


举例:

if !instance_exists(obj_Exit)
   {
   show_debug_message("Exit not created!");
   game_end();
   }

上述代码检查一个实例是否存在，如果不是，则一个调试信息将被显示在编译窗口中，并且游戏会被结束。

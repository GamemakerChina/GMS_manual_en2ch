get_string

显示一个弹出信息以供用户输入一个字符串。

语法:

get_string(str, def);


参数 描述
str 弹出信息中显示的字符串
def 文本框中的默认字符串


返回值: 字符串


描述

这创建了一个弹出窗口显示一个标准消息，带有一个标签为"Ok"的按钮，来提示用户输入一个字符串。这个函数将会返回被输入的字符串，或者如果没有东西被输入①返回默认值。


①：此处的“没有东西被输入”指的是弹出对话框后不修改文本框内容，而并非文本框清空，即所返回内容以文本框内容为准。


注意: 这个函数只用于调试用途。若你想在你最终游戏中要求这个功能，请使用get_string_async。


举例:

global.test_name = get_string("Test highscore name:", "Anonymous");

上述代码将会提示用户给出一个名字，其将接下来会被存入全局变量"test_name"中。如果没有东西被输入并且用户只是按下了"OK"那么默认值，"Anonymous"，将会被返回。

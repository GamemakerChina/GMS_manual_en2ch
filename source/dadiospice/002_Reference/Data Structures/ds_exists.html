ds_exists

Check to see if a data structure of a given type exists.检测一个被给定类型的一个数据结构是否存在。

Syntax语法:

ds_exists(ind, type);


Argument参数 Description描述
ind The variable index to check for the data structure.待进行 数据结构检测的变量索引。
type The type of data structure to check for (see the list of constants below)待检测的数据类型（参见下文的常量表）


Returns返回值: Boolean布尔型


Description描述

With this function you can check to see if a data structure of the given type exists. You supply the "index" value (as held in a variable) and the ds "type", which can be any of the constants listed below, and the function will return true if the data structure exists and false otherwise. 使用这个函数你能够检测观察一个属于该类型的数据结构是否存在。你提供 这个“索引”的值（被承载给一个变量）以及这个数据结构的“类型”，可以是 下方常量列表中的任意一个，并且如果这个数据结构存在这个函数将会返回 true（真），否则返回false（假）

Constant常量 Description描述
ds_type_map A map data structure.一个比对数据结构。
ds_type_list A list data structure.一个列表数据结构。
ds_type_stack A stack data structure.一个堆栈数据结构。
ds_type_grid A grid data structure.一个栅格数据结构。
ds_type_queue A queue data structure.一个队列数据结构。
ds_type_priority A priority data structure.一个优先队列数据结构。


Example举例:

if !ds_exists(ai_grid, ds_type_grid)
   {
   ai_grid = ds_grid_create(room_width / 32, room_height / 32);
   }

The above code checks the (previously initialised) variable "ai_grid" to see if it indexes a ds_grid type data structure, and if it does not then it creates one and stores its index in the variable. 上述代码检测变量"ai_grid"（需要提前初始化）来检测它是否索引着 一个栅格类型的数据结构，并且如果它不是，那么它就创建一个并把它的索 引存给这个变量。

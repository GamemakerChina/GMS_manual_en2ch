ds_grid_multiply

Multiplies the value of a given grid cell by a given amount.
将被给定的栅格的单元格中的值和一个被给定的值相乘。

Syntax 语法:

ds_grid_multiply(index, x, y, val);


Argument参数 Description描述
index The index of the grid.栅格数据的索引
x The x position of the cell in the grid.栅格中单元格的x位置。
y The y position of the cell in the grid.栅格中单元格的y位置。
val The value to multiply with the cell.待与单元格相乘的值。


Returns 返回值: N/A 无


Description 描述

This function will multiply the value of a the given grid cell by the specified amount.
这个函数将会用一个特定的值乘以栅格中被给定的单元格的值。

NOTE 注意: This function will only work with real numbers, not strings.
这个函数将只对实数生效，而不是字符串。


Example 举例:

ds_grid_multiply(mygrid, 5, 5, 2);

the above code will multiply the value stored in the given ds_grid cell by 2.
上述代码将会把存储在被给定栅格单元格的值乘以2

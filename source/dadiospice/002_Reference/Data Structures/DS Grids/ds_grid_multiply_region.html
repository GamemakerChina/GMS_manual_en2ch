ds_grid_multiply_region

Multiplies all the values found in the given region by a given amount.
将在被给定区域内找到的所有值乘以一个被给定的值。

Syntax 语法:

ds_grid_multiply_region(index, x1, y1, x2, y2, val);


Argument参数 Description描述
index The index of the grid.栅格数据的索引
x1 The x position of the left of the region in the grid.矩阵中这个区域的左端x位置。
y1 The y position of the top of the region in the grid.矩阵中这个区域的顶端y位置。
x2 The x position of the right of the region in the grid.矩阵中这个区域的右端x位置。
y2 The y position of the bottom of the region in the grid.矩阵中这个区域的底端y位置。
val The value to multiply with the region cells.待与区域内单元格相乘的值。


Returns 返回值: N/A 无


Description 描述

With this function you can specify a region of the grid in which to multiply each cell value by a given amount.
使用这个函数你能够指定一片每个单元格都要和被给定的值相乘的区域。

NOTE 注意: This function will only work with real numbers, not strings.
这个函数这个函数将只对实数生效，对字符串无效。


Example 举例:

ds_grid_multiply_region( mygrid, 5, 5, 10, 10, 2 );

The above code will take all the values found within the defined rectangular grid area and multiply each one by 2.
上述代码将会取得在矩阵区域内被定义的矩形区域中的所有值并逐一乘以2.

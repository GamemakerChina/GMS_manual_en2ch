ds_grid_add_grid_region

Adds the contents of the cells in a source grid to the cells in a destination grid.
将一个源栅格中包含的单元格叠加给靶栅格中。

Syntax语法:

ds_grid_add_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);


Argument参数 Description描述
index The index of the destination grid.靶栅格的索引
source The index of the source grid.源栅格的索引
x1 The left position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的左端位置。
y1 The top position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的顶端位置。
x2 The right position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的右端位置。
y2 The bottom position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的底端位置。
xpos The x position on the destination grid to add the source region to.靶栅格上待叠加源栅格区域的x位置。
ypos The y position on the destination grid to add the source region to.靶栅格上待叠加源栅格区域的y位置。


Returns返回值: N/A无

This function can be used to add all the values of all the cells found within the source area of a grid to the values within the destination grid, as illustrated below:

你也能使用这个函数给同一个的栅格中将来自于被存储在一片区域的值叠 加给另一片区域（参见下方的代码举例）：


Example举例:

ds_grid_add_grid_region(grid, grid, 0, 0, 1, 5, 2, 0);

The above code would copy the region of cells from (0,0) to (1,5) of the ds-grid indexed in the variable "grid" and add them to the cells from position (2,0) of the same ds_grid.
上方代码能够拷贝索引为变量"grid"的栅格数据中从(0,0)到(1,5)的区域中单 元格的值并将其叠加给同一栅格数据中(2,0)的位置。

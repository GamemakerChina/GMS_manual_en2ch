ds_grid_multiply_disk

Multiply all the values contained within the disc region by a given amount.
将包含于圆盘区域内的所有值都乘以被给定的值

Syntax 语法:

ds_grid_multiply_disk(index, xm, ym, r, val);


Argument参数 Description描述
index The index of the grid.栅格结构的索引。
xm The x position of the disk on the grid.栅格中圆盘的x位置
ym The y position of the disk on the grid.栅格中圆盘的y位置
r The radius of the disk on the grid.栅格中圆盘的半径
val The value to multiply the cells within the disk with.待与圆盘内单元格相乘的值。


Returns 返回值: N/A 无


Description 描述

This function will take all the values in a given disc-shaped region of the ds_grid, and multiply each one by the given amount.
这个函数将取得栅格结构中被给定圆盘形状区域内的所有值并逐个乘以被给定的值。

NOTE 注意: This function will only work with real numbers, not strings.
这个函数这个函数将只对实数生效，对字符串无效。


Example 举例:

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2);

The above code will take all the values found within the circular grid area and multiply each one by 2.
上述代码将会取得在栅格中圆盘区域内被找到的值并逐一乘以2。

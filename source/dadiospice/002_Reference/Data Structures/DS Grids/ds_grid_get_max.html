ds_grid_get_max

Returns the max value from all the cells within a region on a grid.
返回一个矩阵中一片区域内所有单元格的最大值

Syntax语法:

ds_grid_get_max(index, x1, y1, x2, y2);


Argument参数 Description描述
index The index of the grid.栅格数据的索引
x1 The left cell column of the region.区域的左端单元格列。
y1 The top cell row of the region.区域的顶端单元格行
x2 The right cell column of the region.区域的右端单元格列
y2 The bottom cell row of the region.区域的底端单元格行


Returns返回值: Real or String实数型 或者 字符串


Description描述

This function can be used to find the maximum value for all the cells found within the defined region of a grid, as shown in the image below:
这个函数可以被用于在一个栅格数据中被定义的区域内寻找所有单元格的最 大值，正如下图所示：


Example举例:

val = ds_grid_get_max(grid, 0, 0, 5, 5);

The above code will set the variable "val" to the maximum value contained within the given region of the ds_grid indexed in the variable "grid".
上述代码将设定变量"val"为索引为变量"grid"的栅格数据中被给定区域内所 包含的最大值

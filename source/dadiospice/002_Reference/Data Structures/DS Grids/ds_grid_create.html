ds_grid_create

Creates a new grid.
创建一个新的栅格结构。

Syntax语法:

ds_grid_create(w, h);


Argument参数 Description描述
w The width of the grid to be created.待创建的栅格结构的宽度。
h The height of the grid to be created.待创建的栅格结构的高度


Returns返回值: Real实数型.


Description描述

With this function you can create a new ds_grid data structure of the specified cell width and height. This function returns an id which must be used in all further functions that deal with this ds_grid.
使用这个函数你能够创建一个新的指定单元格宽度和高度的栅格数据结构。 这个函数返回一个在所有更深层的处理这个栅格结构的函数中必须被使用的id。


Example举例:

mygrid = ds_grid_create(10, 10);

This creates a grid 10 cells high and 10 cells wide.
这将创建一个10个单元格高且10个单元格宽的栅格结构。

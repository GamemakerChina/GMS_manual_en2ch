ds_grid_multiply_grid_region

Multiplies the contents of the cells in a source grid with the cells in a destination grid.
将源栅格中包含的单元格与靶栅格中单元格相乘。

Syntax 语法:

ds_grid_multiply_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);


Argument参数 Description描述
index The index of the destination grid.靶栅格的索引
source The index of the source grid.源栅格的索引
x1 The left position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的左端位置。
y1 The top position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的顶端位置。
x2 The right position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的右端位置。
y2 The bottom position of the region of cells to copy from the source grid.从源栅格中待拷贝单元格的底端位置。
xpos The x position on the destination grid to multiply the source region with.靶栅格上待相乘源栅格区域的x位置。
ypos The y position on the destination grid to multiply the source region with.靶栅格上待相乘源栅格区域的y位置。


Returns 返回值: N/A 无


Description 描述

With this function you can define an area within a given ds_grid, then take those values and multiply them with those found in a separate region of either the same ds_grid, or another one (which has been previously created). The original region will remain unchanged, while the region that they have been multiplied with will now store the new values for each cell.
使用这个函数你能够在一个被给定的栅格数据中定义一片区域， 然后取得这些值并将其与另一片要么是相同栅格数据中要么是另一个栅格（已经被提前创建过）相乘。 原始区域将保持不变，而被与之相乘的区域此时在每个单元个将会存入新的数值。


Example 举例:

ds_grid_multiply_grid_region(mygrid, mygrid, 0, 0, 5, 5, 0, 0);

This would take the region of cells from (0,0) to (5,5) of the ds_grid "mygrid" and multiply them with the cells from position (0,0) of the same ds_grid.
这将会取得栅格"mygrid"中从(0,0)到(5,5)区域的单元格并将其乘给同一个栅格上单元格从(0,0)开始的区域。

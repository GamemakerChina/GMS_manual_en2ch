ds_grid_add

Adds a value to a cell in a grid.将一个值叠加到栅格的一个单元格中。

Syntax语法:

ds_grid_add(index, x, y, val);


Argument参数 Description描述
index The index of the grid.栅格结构的索引
x The x position of the cell in the grid.栅格中单元格的x位置。
y The y position of the cell in the grid.栅格中单元格的y位置
val The value to add to the cell.待叠加到栅格中的值


Returns返回值: N/A无


Description描述

This function can be used to add a given value (real or string) to the value of the given cell within the grid. The value to be added must be the same type as that held within the grid cell, ie: you cannot add a string to a real or vice-versa, and for strings this corresponds to concatenation. 这个函数能够被用于叠加一个被给定的值（实数或字符串）到栅格中被 给定的单元格中。这个值必须和单元格内承载的值的类型相同，比如说：你 不能把一个字符串叠加给一个实数值，反之亦然，并且对于字符串来说这将 对应到串接字符串的位置。


Example举例:

ds_grid_add(grid, 5, 5, 6);

This would add 6 to the given cell within the ds_grid referenced by the variable "grid".这将把6叠加到被给定的在被变量grid引用的ds_grid 中的单元格

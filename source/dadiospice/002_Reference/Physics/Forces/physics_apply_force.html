physics_apply_force

这个函数对当前实例在房间内一个位置以由一个向量定义的强度施加一个力。

语法:

physics_apply_force(xpos, ypos, xforce, yforce)


参数 描述
xpos 力将被施加的房间内的x坐标
ypos 力将被施加的房间内的y坐标
xforce 力矢量的x分量
yforce 力矢量的y分量


返回值: 无


描述

当用物理工作时你不仅需要对物体施加重力，还要施加其他外力。一个作用力是引发物体经历在速度、方向或形状方面改变的任何影响并且在GameMaker: Studio中和我们定义重力一样被定义，通过从xforce/yforce分量中得到矢量。需要注意的是，外力不被施加于物体的质心，而是房间内的一个位置并且它们将会改变方向、速度和依赖于施加在这个物体上的其他力（像重力）的转矩（转动）。这里有一个例证图片来帮助理解：

在上述范例中底端的弹簧物体在球物体上创建了一个力来使它向另一个方向移动。实质上的力被我们从在关于xpos/ypos位置x/y分量中获得的矢量所定义的，也就是说力被计算为从(xpos,ypos)到(xforce,yforce)的距离以N（牛顿，1N被定义为将一个1kg物体以1m/s2的变化率加速所需要的净力值）为单位并且这个方向是我们由从(xpos,ypos)到(xforce,yforce)的方位角。这段类似于的碰撞代码将会参与到弹簧物体的碰撞事件中并且将会看起来像下面被提供的举例中那样。

注意:物理世界使用单位m进行所有计算，并且像素/米的比率被定义于房间编辑器中或使用physics_world_create.


举例:

with (other)
   {
   physics_apply_force(x, y, 0, -30);
   }

上述代码将会对在碰撞中的other物体应用一段代码带有一个30N的力和方向。

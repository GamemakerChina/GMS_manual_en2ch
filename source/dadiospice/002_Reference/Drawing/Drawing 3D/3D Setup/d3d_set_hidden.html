d3d_set_hidden

开启或者关闭深度测试

语法：

d3d_set_hidden(enable)


参数 说明
enable 开启 (true) 或者关闭 (false) 深度测试


返回值： N/A


说明

默认情况下，将需要绘制的新像素的z值与深度缓冲区中对应位置的z值进行比较，如果比深度缓存中的值小，那么用新像素的颜色值更新帧缓存中对应像素的颜色值。


示例：

d3d_set_hidden(true);

开启深度测试

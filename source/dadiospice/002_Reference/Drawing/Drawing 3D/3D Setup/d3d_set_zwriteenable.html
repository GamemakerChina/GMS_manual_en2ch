d3d_set_zwriteenable

开启 或 关闭 深度缓存。

语法：

d3d_set_zwriteenable(enable)


参数 说明
enable 开启 (true) 或者 关闭 (false) 深度缓存。


返回值： N/A


说明

这个函数能够开启 (true) 和关闭 (false) 缓存，以便进行深度测试。默认开启。

3D模式下的绘制，屏幕上的每个像素都存在一个深度值，被存在深度缓存中。每次绘制会进行深度测试，新的深度会与当前位置缓存的深度值进行比较，并更新为深度最小的。 并非任何情况下，深度缓存是我们所需要，你可以关闭它。


示例：

d3d_set_zwriteenable(false);

以上代码关闭深度缓存，影响之后的所有绘制

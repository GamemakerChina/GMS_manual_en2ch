d3d_set_culling

开启 或 关闭 背面剔除

语法：

d3d_set_culling(enable)


参数 说明
enable 开启 (true) 或 关闭 (false) 背面剔除


返回值： N/A


说明

一个三角形图元，具有前、后两面，顶点按逆时针排列的那一面为正面（表面的法向量的正方向）。默认情况下，两个面都会被绘制。 当你绘制一个密封的实体时，背面的绘制将是相当的浪费，而且我们看不到背面，比如正方体内部的面，在开启背面剔除后，你跑进这个正方体内，你会发现正方体消失了。


示例:

d3d_set_culling(true);
d3d_primitive_begin(pr_trianglelist);
d3d_vertex(100, 100, 0);
d3d_vertex(100, 200, 0);
d3d_vertex(150, 150, 200);
d3d_vertex(100, 200, 0);
d3d_vertex(200, 200, 0);
d3d_vertex(150, 150, 200);
d3d_vertex(200, 200, 0);
d3d_vertex(100, 100, 0);
d3d_vertex(150, 150, 200);
d3d_vertex(100, 100, 0);
d3d_vertex(100, 200, 0);
d3d_vertex(200, 200, 0);
d3d_primitive_end();

绘制一个金字塔，背面是被剔除的。你跑进这个金字塔内，你会发现金字塔消失了。

3D 绘图

这部分详解 3D 图元、几何的绘制.

3D绘制，我们要注意：当我们在2D模式下绘制一个三角形的时候，三个角（顶点）的深度（depth）是一样的。然而，3D 模式下的加入了一个深度值，我们通常叫做Z轴。这时候，我们用（x,y,z)来表示一个3D空间的顶点。此外，我们可以使用draw_set_colour and draw_set_alpha 来改变将绘制的物体的颜色和透明度。

可用的3d图元基本函数如下：

  1. d3d_primitive_begin
  2. d3d_vertex
  3. d3d_vertex_colour
  4. d3d_primitive_end

对于带纹理的3D图元如下 (结束本批次可以使用上面提到的函数 d3d_primitive_end()):

  1. d3d_primitive_begin_texture
  2. d3d_vertex_texture
  3. d3d_vertex_texture_colour

以下是一些内置的预先定义好的几何体，你可以设置是否带纹理，并且3D相关的“环境参数”也能影响这些几何体的绘制，比如backface culling 开启背面剔除):

  1. d3d_draw_block
  2. d3d_draw_cylinder
  3. d3d_draw_cone
  4. d3d_draw_ellipsoid
  5. d3d_draw_wall
  6. d3d_draw_floor

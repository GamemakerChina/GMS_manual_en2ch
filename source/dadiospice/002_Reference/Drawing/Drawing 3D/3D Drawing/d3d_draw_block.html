d3d_draw_block

绘制一个立方体。

Syntax:

d3d_draw_block(x1, y1, z1, x2, y2, z2, tex, hrepeat, vrepeat)


参数 说明
x1 起点X。
y1 起点y。
z1 起点z。
x2 终点x
y2 终点y
z2 终点z
tex 纹理id (-1 表示不使用纹理)
hrepeat 纹理水平方向平铺的次数
vrepeat 纹理垂直方向平铺的次数


返回值： N/A


说明

这个函数绘制一个立方体，参数中的起点位置和终点位置的两个角是对角。如果你不打算使用纹理，可以设参数tex为-1， 同时，参数hrepeat、vrepeat将会被忽略。

注意：只有当你打算平铺纹理质的时候，纹理的规格才必须为2的N次方（比如：8x8, 128x128, 256x256）


示例：

var tex;
tex = background_get_texture(bck_Wall);
d3d_draw_block(20, 20, 20, 80, 40, 200, tex, 1, 1);
d3d_draw_block(200, 300, -10, 240, 340, 100, tex, 1, 1);

以上代码，把 bck_Wall 的纹理 id 存入变量"tex"，并绘制了两个带有此纹理的立方体

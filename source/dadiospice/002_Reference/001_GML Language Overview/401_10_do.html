do 语句

这一节将学习“do”语句。

“do”也就是“do... until”，缺一不可。结构如下：

do <语句> until (<表达式>)


语句（也可以是代码块）会一直运行直到表达式为真，至少会运行一次。注意这是一个循环，可能会出现死循环，游戏将会挂起不响应输入。下面是“do... until”的一个经典例子：

{
do
   {
   x = random(room_width);
   y = random(room_height);
   }
until (place_free(x, y)); }


上述代码尝试去在一个空闲位置放置当前的实例（这和动作 移动物体至随机位置结果相似）。

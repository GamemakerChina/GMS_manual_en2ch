"if"语句

这节概述了如何使用"if"语句。

一个简单的"if"语句具有这种形式：

if (<expression>) <statement>


或者它也可以有略微复杂的"if... else..."形式：

if (<expression>) <statement> else <statement>


在这个情况下，这个表达式（expression）将被评估，并且如果（近似）值是<=0(false)，语句后面的else被执行，否则(true)另一个语句被执行。经常在if后面的语句周围放上花括号是一个好习惯，并且为每一个语句另起一行，所以最终代码将会有这种格式：

if (<expression>)
   {
   <statement>
   ...
   <statement>
   }
else
   {
   <statement>
   }


作为一个小的范例，考虑下面这段把实例朝向房间内x=200位置移动的代码：

if (x < 200)
   {
   x += 4;
   }
else
   {
   x -= 4;
   }

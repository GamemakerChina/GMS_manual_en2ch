repeat 语句

这节解释如何使用“repeat”。

“repeat”语句的结构：

repeat (<表达式>) <语句>


语句（statement）被重复执行的次数由表达式（expression）的近似值指定。比如说，下面的程序将会在随机位置创建5个球。

{
repeat (5) instance_create(random(400), random(400), obj_ball);
}


这对于规避键入多次相同的代码、或者使用数组、又或者一些操作中的计数等等是非常有效的。比如：

{
var i, total;
i = 0;
total = 0;
repeat (10)
   {
   total += array[i];
   i += 1
   }
draw_text(32, 32, total);
}

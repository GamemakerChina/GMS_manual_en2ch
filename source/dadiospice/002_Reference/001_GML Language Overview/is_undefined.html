is_undefined

检测参数的数据类型，是否为undefine。

语法：

is_undefined(n);


参数 说明
n 被检测的值


返回： 布尔值


说明

这个函数用于检测参数的数据类型，是否为undefine。 更多与数据类型相关的信息， 可在这里查看。


示例：

var val = ds_map_find_value(map, 13);
if is_undefined(val)
   {
   show_debug_message("Map entry does not exist!");
   }

以上代码，检测变量“val”的数据类型是否为 undefine 。如果为真，则弹出对话框。


返回：Real Number Functions

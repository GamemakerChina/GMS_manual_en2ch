"while"语句

这节将处理"while"语句及其使用。

一个"while"语句具有这种形式

while (<expression>) <statement>


只要表达式（expression）是true，这些语句（statement）（也可以是语句块）就会被反复执行。请小心你的while循环！你很容易就能造成无限循环，在这种情况下你的游戏将被挂起并且不再响应任何的用户输入。下文你能找到一个典型使用"while"的方式的一个范例：

{
while (!place_free(x, y))
   {
   x = random(room_width);
   y = random(room_height);
   }
}


上述代码尝试去在一个空闲位置放置当前的实例 （这和move an object to a random position动作几乎等效）。

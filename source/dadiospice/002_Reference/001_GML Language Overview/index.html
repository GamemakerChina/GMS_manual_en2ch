GML 概述

GameMaker Language（简称GML），是GameMaker: Studio 的专有编程语言。

GML相当的灵活，是拖按钮所不及的。比如：

  • 脚本：一段代码，也是常说的函数。
  • 事件：在对象事件中加入GML代码，你能够更好的控制这个对象的行为，或者与其他对象的交互。此外，GML代码块能与按钮混用！

  • 房间创建时脚本：每次房间被创建的时候，程序就会触发带有GML片段的特殊事件：“创建事件”。

  • 实例创建时脚本：正如你在对象的创建事件中加入的代码片段，你也可以给放置在房间中的对象实例，加入特有的脚本。实例创建时事件在对象创建时事件之 后执行。(之后的教程 中进行详解）。

本章节讲解GML的语法知识，告诉你如何编写自己的脚本，并控制游戏逻辑。
注意：GM使用者的资源名的命名潜规则，比如精灵资源开头为“spr_”。更多细节，你会在实际项目或者教程中看到。
你可以搜索整个项目中的脚本，找出指定的字符串的位置。相关细节可查阅 Scripts 。

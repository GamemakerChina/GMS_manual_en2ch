audio_system

检查正在使用的是传统声音系统还是常规声音系统

语法:

audio_system();


返回值: 常量


描述

这个函数返回当前的声音系统是否是传统的还是常规的。返回值可以是下列两个常数之一：

常量 描述
audio_old_system 正在使用传统声音系统
audio_new_system 正在使用常规声音系统


如果你使用的是html5的目标模块，那么这个函数非常重要。一些浏览器无法与新声音系统兼容，因此Gamemaker:Studio将会自动换回传统声音引擎。


例子:

if audio_system() == audio_old_system
   {
   global.SS = false;
   }
else
   {
   global.SS = true;
   }

上面的代码检查所使用音响系统的是传统的或新的之中的哪一个，并相应地设置全局变量。

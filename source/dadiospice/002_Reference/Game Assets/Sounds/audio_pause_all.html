audio_pause_all

暂停所有的声音。

语法:

audio_pause_all();


返回值: N/A


描述

用这个函数你可以暂停所有正在播放的声音。


例子:

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_all();
      }
   else
      {
      audio_resume_all();
      }
   }

上述代码检查按键"P"是否被按下，如果是，则将全局变量Pause设置为true或false， 然后暂停或重启之前被暂停的音乐

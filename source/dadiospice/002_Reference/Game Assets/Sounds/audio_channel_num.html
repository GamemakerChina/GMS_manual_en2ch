audio_channel_num

设定可用声道的数目

语法：

audio_channel_num(num);


参数 描述
num 可用声道数 （默认为128）.


返回值: N/A


描述：

用这个函数你可以设定在Gamemaker:Studio中有多少可以用来播放声音的声道数。也就是你可以同时播放声音的数目， 并且如果超过了设定量，低优先级的声音将会被停止播放，以腾出声道给高优先级的音乐。你可以用此函数来优化你的游戏，以更少的的声音通道换取更高的性能，但要记住这也会在大量的声音同时播放时，造成一些声音被阉割掉。

例子：

switch (os_browser)
   {
   case browser_not_a_browser:
      switch (os_type)
         {
         case os_windows:
         case os_macos:
            audio_channel_num(200);
            break;
         default:
            audio_channel_num(64);
            break;
         }
      break;
   default:
      audio_channel_num(16);
      break;
   }

上述代码检查运行该游戏的平台，并改变可用声道数，以提高性能。

audio_exists

检查给定的声音是否存在。

语法:

audio_exists(index);


参数 描述
index 用来检查是否存在的声音的索引。


返回值:布尔


描述:

这个函数返回一个声音是否存在（true or false）。请注意，如果您调用索引以前没有初始化，这个函数会导致错误,因为它调用了不存在的资源索引.要检查的声音可以是单个的声音实例(使用audio_play_sound或audio_play_sound_at函数时,存储在一个变量的单个声音) 或者是一个声音资源


例子:

if audio_exists(global.Music)
   {
   audio_play_music(global.Music, true);
   }

上述代码检查一个声音是否存在。如果是,则它将被循环播放。

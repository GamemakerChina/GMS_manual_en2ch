audio_pause_music

暂停指定的音乐（淘汰）。

Syntax:

audio_pause_music();


返回值: N/A


描述:

注意! 这个功能已经过时，你应该使用audio_pause_sound代替。

使用此函数，你可以暂停当前正在播放的音乐。


例子:

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_music();
      }
   else
      {
      audio_resume_music();
      }
   }

上述代码检查按键“P”是否被按下， 并且如果检测到，则将全局变量“Pause”设置为true，然后根据“Pause”的值暂停或恢复音乐

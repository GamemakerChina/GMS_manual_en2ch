audio_resume_sound

恢复被暂停的指定音乐。

语法:

audio_resume_sound(index);


参数 描述
index 被恢复的声音索引


返回值: N/A


描述

有了这个函数，你可以恢复当前暂停的任何声音（使用函数audio_pause_sound后）。可以是一个单一的声音实例 （当你使用audio_play_sound或audio_play_sound_at函数时，可以将被播放的声音索引存储在一个变量） 或是一个原声资源，但在这种情况下，给定声音的所有实例将被重启。



例子:

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_sound(snd_Waterfall);
      }
   else
      {
      audio_resume_sound(snd_Waterfall);
      }
   }

上述代码检查按键"P"是否被按下，如果是，则将全局变量Pause设置为true或false， 然后将变量“snd_Waterfall”中的声音暂停或恢复。

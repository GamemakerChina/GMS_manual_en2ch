audio_get_name

返回给定原声资源的名称（字符串）。

语法:

audio_get_name(index);


参数 描述
index 被检查的声音索引


返回值: 字符串

描述

该函数将以字符串形式返回一个给定的音频资源的名称。"index"值可以是资源自身（就像你在资源树里看到的那样），或者当您播放声音时被赋予唯一的ID值，例如audio_play_sound(). 注意，返回的字符串与资源ID不一样，并且不能用来访问资源本身，所以应当只用于显示或错误检查。


例子:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

上述代码从三个不同的声音资源中随机选择一个播放，接着显示一个带有其名称的调试信息

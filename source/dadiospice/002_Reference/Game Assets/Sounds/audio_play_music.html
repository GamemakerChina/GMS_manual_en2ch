audio_play_music

将指定背景音乐播放一次或循环播放

Syntax:

audio_play_music(index, loop);


Argument Description
index 播放背景音乐的索引
loop 标志音乐是否循环播放


返回值: N/A


描述

注意! 这个函数已经淘汰，你应该使用audio_play_sound和audio_stop_sound代替。

这个函数常常被用来开始播放一段背景音乐。当你使用此函数时只能依次播放一段背景音，开始播放一段新音乐时将直接结束前一个正在播放的音乐，您还可以选择将loop标记为true使音乐循环播放，或者设置为false令其仅播放一次


Example:

if !global.Music
   {
   audio_stop_music();
   }
else
   {
   audio_play_music(snd_MusicIntro, true);
   }

上述代码检查全局变量“Music”，如果返回false，则停止所有正在播放的背景音乐，如果返回true，它会循环播放在变量“snd_MusicIntro”中的音乐。

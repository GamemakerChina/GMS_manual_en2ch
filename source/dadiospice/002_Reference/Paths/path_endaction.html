path_endaction

Sets what to do when the instance hits the end of the path.

Syntax:

path_endaction;


Returns: N/A.


Description

This function can be used to change the reaction of an instance when it reaches the end of the current path. Normally you would set this when you start the path using path_start but you may wish to change this behaviour depending any number of events in your game. The available values are:

  • 0 = stop the path
  • 1 = continue from start position, jumping to the start if the path is not closed
  • 2 = start the path again from the current position
  • 3 = reverse the speed of the path (run the path in reverse)


Example:

path_endaction = irandom(3);

The above code will change the path end action randomly to any of the three possible choices.

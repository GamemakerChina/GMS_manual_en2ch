arcsin

Returns the inverse sine of x.
返回x的反正弦值

Syntax 语法:

arcsin(x);


Argument参数 Description描述
x The angle (in radians) to return the inverse sine of.待转换为反正弦值的角（弧度制）


Returns 返回值: Real 实数型


Description 描述

Returns the inverse sine of x, in that if sin(x)=n, arcsin(n)=x, and the resulting number will be between -pi/2 and pi/2.
返回x的反正弦，换句话说，如果sin(x)=n，那么arcsin(n)=x，并且结果将是一个-π/2到π/2的数。

NOTE 注意: This will only accept a value between -1 and 1. (anything else will throw an error).
这只接受在-1到1之间的值①。（其它东西的话会抛出错误）

①arcsin的定义域为[-1,1]。


Example 举例:

val = arcsin( 0 );

This will set val to 0.这将把val设置为0.

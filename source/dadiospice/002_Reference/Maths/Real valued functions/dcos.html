dcos

返回val的余弦值，其中val是角度制的。

语法:

dcos(val);


参数 描述
val 待转换为余弦值的角（角度制）。


返回值: 实数型


描述

在直角三角形中 cos(val) = 邻边 / 斜边 其中val是三个角之一。这个函数总是返回一个在[-1,1]上的数。 This function


举例:

val = dcos(45);

这将把"val"设定为0.71①。

①：√2/2≈0.71

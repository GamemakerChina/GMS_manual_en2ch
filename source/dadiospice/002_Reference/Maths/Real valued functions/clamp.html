clamp

Returns a value clamped between the specified minimum and maximum.
返回一个被限定在被指定的最小值和最大值之间的数

Syntax 语法:

clamp(val, min, max)


Argument参数 Description描述
val The value to clamp.待限定的值
min The minimum value to clamp between.待限定的最小值
max The maximum value to clamp between.待限定的最大值


Returns 返回值: Real. 实数型


Description 描述

With this function you can maintain an input value between a specified range.
使用这个函数你能够把一个输入值维持在一个指定范围之间。


Example 举例:

speed = clamp(speed, 1, 10);

The above code will clamp the speed so that it never falls below 1 or goes over 10.
上述代码将限定速度以便于它不会跌到1以下也不会涨到10以上。

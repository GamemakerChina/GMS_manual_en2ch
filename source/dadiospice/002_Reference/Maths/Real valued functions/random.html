random

返回在[0,n)上的一个随机实数。

语法:

random(n);


参数 描述
n 待选择随机数的范围的上边界。


返回值: 实数型


描述

这个函数对于不必需要整数的概率是很好的， 举个例子, random(100)将会返回0到99的值①，但值将可能会是22.56473！ 你也可以在这个函数中不是用整数而使用实数像这样——random(0.5)，将会返回一个从0到0.4999999的一个数值.


①：根据后文内容以及实际测试，此处表述不当，应为返回在[0,100)上的值，即这个值有可能大于99而小于100.
注意:这个函数在每次游戏重新运行时将返回相同的数因为事实上GameMaker: Studio 每次都产生相同的初始随机数种子以使得调试代码成为相当容易的事情。欲避免这个行为请在游戏开始时使用randomize


举例:

if random(10) >= 9
   {
   score += 100;
   }

这将会近似产生1/10的几率为分数增加100。

arccos

Returns the inverse cosine of x.
返回x的反余弦值

Syntax 语法:

arccos(x)


Argument参数 Description描述
x The angle (in radians) to return the inverse cosine of.
待转换为反余弦值的角（弧度制）


Returns 返回值: Real 实数型


Description 描述

Returns the inverse cosine of x, in that if cos(val)=n, arccos(n)=val, and the resulting number will be between pi and 0.
返回x的反余弦值，换句话说如果cos(val)=n则arccos(n)=val，并且结果将会是π到0之间的数。

NOTE 注意: This will only accept a number between -1 and 1 (anything else will throw an error).
这将只接受在-1到1之间的数①（其他东西的话会抛出一个错误）

①arccos定义域为[-1,1]


Example 举例:

val = arccos(0);

This will set val to pi/2.
这将会把val设定为π/2

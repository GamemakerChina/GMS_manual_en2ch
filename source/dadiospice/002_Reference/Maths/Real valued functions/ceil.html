ceil

返回被给定的数向上取整最近的整数。

语法:

ceil(val);


参数 描述
x 待转换的数。


返回值: 整数


描述

这个函数能够取得任意的实数并且将其向上取整到最近的整数。你应该小心这个函数的一个常见错误就是使用它来向上取整一个随机数并期望得到的数总是大于1的，比如：

int = ceil(random(5));


现在你将会期待这个代码总是给出一个在1到5之间的整数，但是这可能并不产生这种情况因为有非常小的概率这个随机函数会返回0，并且对0向上取整你还是会得到0. 这是微乎其微的可能性但是应该在使用这个函数时算在考虑范畴之内。


举例:

val = ceil( 3.4 );

这将把val设为4.

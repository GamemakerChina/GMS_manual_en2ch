frac

返回被给定数的小数部分。

语法:

frac(n);


参数 描述
n 待改变的数值。


返回值: Real


描述

这个函数返回n的小数部分，也就是，小数点后的部分。 它将 只会返回小数的小数点后的值， 所以frac(3.125)将返回0.125， frac(6.921)将返回0.921，等等...


举例:

val = frac(3.4);

这将把val设置为0.4.

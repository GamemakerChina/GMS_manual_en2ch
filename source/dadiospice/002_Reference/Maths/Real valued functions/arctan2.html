arctan2

Calculates arctan(y/x) and returns an angle in the correct quadrant.
计算反正切（y/x）并返回正确的象限角。

Syntax 语法:

arctan2(y, x);


Argument参数 Description描述
y The y coordinate to calculate.待计算的y坐标。
x The x coordinate to calculate.待计算的x坐标。


Returns 返回值: Real 实数型


Description 描述

This function returns the inverse tangent of an angle y/x, where y = Opposite side of triangle and x = Adjacent side of triangle. Unlike arctan(x) the function arctan2(y, x) is valid for all angles and so may be used to convert a vector to an angle without risking division by zero, and it also returns a result in the correct quadrant.
这个函数返回一个角y/x的反正切值，其中y是三角形的对边而x是三角形的邻边。不像arctan(x)函数那样，arctan2(y,x)对于任意角度都是合法的，并且可以用于将一个矢量转换为一个角度而无虞除以0的问题，并且他也会返回正确的象限位置。

NOTE 注意: The value returned is in radians, not degrees.
这个返回值是弧度制，而非角度制。


Example 举例:

val = arctan2(1, 1);

This will set val to the correct angle, in this case 0.79.这将把val设为正确的角度，在这里面是0.79（π/4）。

math_set_epsilon

为浮点计算设定"机器ε"（即机器无穷小值）.

语法:

math_set_epsilon(epsilon);


参数 描述
epsilon 新的ε值 (从 0 到 0.999999999).


返回值: 实数型


描述

ε是一个值被用于却决定两个以发生浮点误差的数是否足够接近以至于被认为是“相等的。 这在处理浮点数学的时候是非常有用的因为它能减小使特定的运算符返回不正确的或者与我们预期相反的结果的“浮点误差”。 举个例子，我们可能会向子图索引添加一个值并且期望是一个单整数，但是因为浮点数学的本性， 实际结果以一些像5.0000002的东西而高中，所以当我们使用下面这个检测：

if image_index == 5 {do something}

这段代码并不像与其所表现得那样 并且返回 false. 然而，然而如果我们设定ε值为0.000001，image_index 值将被取整为最近的与原值相差±0.000001的实数使得上述比较返回true

ε值将被用于下列整型运算符中：

  • <: 小于
  • >: 大于
  • ==: 等于
  • <=: 小于等于
  • >=: 大于等于
  • !=: 不等于

注意将ε值设为0则会禁用所有的取舍，而使用1将发生错误。


举例:

math_set_epsilon(0.0001);

这将为接下来所有浮点运算符设定ε值。

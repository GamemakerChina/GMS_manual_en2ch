choose

Randomly chooses one of up to 16 arguments.随机地从16个参数中选出一个

Syntax 语法:

choose(val0, val1, val2... val15);


Argument参数 Description描述
val0... val15 An input value that can be string, integer, variable or constant.一个可以是字符串、整数、变量或者常量的输入值


Returns 返回值: One of the given arguments. 被给定的参数之一


Description 描述

Sometimes you want to specify something other than numbers for a random selection, or the numbers you want are not in any real order or within any set range. In these cases you would use choose() to generate a random result. For example, say you want to create an object with a random sprite at the start, then you could use this function to set the sprite index to one of up to 16 sprites.

有时候你希望指定某些不是数字内容的随即选项，或者这些数字你希望不是以任何一种真实的顺序或者在任意被设定的范围之内。 在这些情况中你将使用choose()来生成一个随机结果。 比如，说你想要创建的物体在开始的时候带有一个随机的精灵，那么你就可以使用这个函数来把精灵索引设置为16个精灵中的一个。
NOTE 注意: This function will return the same value every time the game is run afresh due to the fact that GameMaker: Studio generates the same initial random seed every time to make debugging code a far easier task. To avoid this behaviour use randomize at the start of your game.
这个函数每次重新运行游戏都将返回同一套结果因为GameMaker: Studio每次都生成同一套初始随机数种子来让调试更方便。为了避免这种行为在你游戏开始的时候使用函数randomize.


Example 举例:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40); name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

The above code uses choose to set a number of properties for the instance.
上述代码使用choose来设定这个实例的一系列属性。

darccos

Returns the inverse cosine of val, where val is in degrees.
返回val的反余弦值，其中val是角度制的。

Syntax 语法:

darccos(val)


Argument参数 Description描述
x The angle (in degrees) to return the inverse cosine of.待转换为反余弦值的角（角度制）。


Returns 返回值: Real 实数型


Description 描述

Returns the inverse cosine of x, in that if dcos(val)=n, darccos(n)=val, and the resulting number will be between 180 and 0.
返回x的反余弦值，其中如果dcos(val)=n那么darccos(n)=val，并且返回的值总是在180到0之间。

NOTE 注意: This will only accept a number between -1 and 1 (anything else will throw an error).
这只接受在-1到1之间的值①。（其它东西的话会抛出错误）
①darccos的定义域为[-1,1]。


Example 举例:

val = arccos(-1);

This will set "val" to 180.
这将把"val"设为180.

median

返回最多16个实数参数的中位数（中间值）。

语法:

median(val1, val2, ... val15);


参数 描述
val0 ... val15 待比较的值。


返回值: 实数型


描述

这个函数返回输入值的中位数，也就是中间值。当参数数量是一个偶数时，两个中间值中较小的一个将被返回①并且这个函数最多可以有16个参数且必须都为实数型。 这也就意味着，比如说，median(43, 12, 25, 19, 6) 将返回19，因为它是所有其余值的中间数。


①：数据统计中的中位数与之有一定出入，统计学的中位数在数据数量为偶数个时，取两个中间值的均值


举例:

x = median( 0, x, room_width );
y = median( 0, y, room_height );

这将阻止玩家从房间的任意一边离开，通过将中位数用作一个夹子（clamp）。 如果玩家，比如，移动到房间的左边界，它的x将小于0，这将意味着第一个中位数函数的中间值为0，所以玩家将会跳转到(0,y)。

point_distance_3d

返回由指定的组成点组成的向量的模。

语法:

point_distance_3d(x1, y1, z1, x2, y2, z2);


参数 描述
x1 向量第一个组成点的x坐标
y1 向量第一个组成点的y坐标
z1 向量第一个组成点的z坐标
x2 向量第二个组成点的x坐标
y2 向量第二个组成点的y坐标
z2 向量第二个组成点的z坐标


返回值: 实数


描述

这个函数取得所提供的向量组成点并且返回向量的长度（模）。它和point_distance()的工作原理相同，但是为3D空间加入了z值（深度）因素。


举例:

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
   {
   ex = inst.x;
   ey = inst.y;
   ez = inst.z;
   if point_distance_3d(x, y, z, ex, ey, ez) < 200
      {
      instance_create(x, y, obj_Missile)
      }
   }

上述代码将会获得最近的敌人的x、y和z坐标，并且然后用它们来检测由它们和玩家坐标所形成的向量的模（长度）。如果这个值小于200，玩家物体就会创建"obj_Missile"的实例。

display_set_windows_vertex_buffer_method

设置 Windows 目标平台的几何顶点缓冲区的方法。

语法：

display_set_windows_vertex_buffer_method(method);


参数 描述
method 准备使用的缓冲区方法（查看下面所列的常量）。


返回：N/A


描述

这个只用于 Windows 的函数允许你选择多个不同的处理游戏几何顶点缓冲区的方法中的一个，它们中的一些在不同的机器上可能表现得更好。这个可也可以在全局游戏设定的 Windows 选项卡中设置，根据不同的机器将会有不同的性能表现。可用的常量如下：

常量 描述
vbm_fastvbm_fast 这是默认的，且是Nvidia推荐的方法 - 它应该是最高的性能。
vbm_compatible 这是在采用快速方法之前（vbm_fast），GameMaker: Studio 曾经使用过的一个过时的方法 - 在较新的机器上性能较差。
vbm_most_compatible 这是 GameMaker 之前使用的一个传统方法 - 性能不如上面两个，但是应该兼容几乎任何一台机器。


举例：

if display_get_windows_vertex_buffer_method() != vbm_fast
   {
   display_set_windows_vertex_buffer_method(vbm_fast);
   }

上述代码将检测当前的几何顶点缓冲区的方法，如果没有准备好则设置为 “fast（vbm_fast 方法）”。

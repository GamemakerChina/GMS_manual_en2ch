display_get_width

返回显示器的宽度。

语法：

display_get_width();


返回： 实数


描述

这个函数将返回显示器的宽度（以像素为单位）。需要注意的是HTML5平台，这个值返回的是 浏览器窗口的宽度，而不是显示器的物理尺寸。


举例：

mywidth = display_get_width();

设置变量“mywidth”为显示器的宽度。


返回: 窗口和视野

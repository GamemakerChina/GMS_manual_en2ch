view_current

返回当前正在渲染的视野。

语法：

view_current


返回： 实数


描述

这个 只读 变量只在Draw Event 中有效，并返回当前被渲染的视野。当存在多个视野时，每个视野将在绘制事件中依次调用一次，这个变量的返回值也会在绘制时同时改变。所以当（比如）你在游戏房间中使用view[0] 和 view[1]，所有实例的绘制事件将运行两次，每个视野一次，通过这个变量可以检测出当前正在绘制哪个视野。一般情况下，当多个视野同时可见时，这个变量只用来渲染某个视野的特定细节。请看下面示例代码：


举例：

if view_current = 0
   {
   draw_text(view_xview[0] + 32, view_yview[0] + 32, "Player 1");
   }
else
   {
   draw_text(view_xview[1] + 32, view_yview[1] + 32, "Player 2");
   }

上述代码将基于返回值检测当前绘制的是哪个视野并绘制不同的文字。

window_view_mouse_get_y

返回相对于给定视野的鼠标x坐标。

语法：

window_view_mouse_get_y( id );


参数 描述
id 比较鼠标位置的视野的id。


返回： 实数


描述

这个函数将返回相对于所选视野的鼠标y坐标。下面的图片说明了工作原理。




举例：

if mouse_check_button_pressed(mb_left)
   {
   var xx, yy;
   xx = window_view_mouse_get_x(0);
   yy = window_view_mouse_get_y(0);
   if xx > 0 && xx < 32 && yy > 0 && yy < 32
      {
      b_press[0] = true;
      }
   }

上述代码将检测鼠标是否按下，如果是，然后它会获取相对于视野0（view[0]）的鼠标位置，并且比较它的值，看是否应该设置为真。

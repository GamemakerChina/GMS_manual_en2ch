view_hport

绘制区域的视口高度。

语法：

view_hport[0...7]


返回： 实数


描述

这个变量可以用来获取或设置特定视口的高度。视口高度（有一个以上的可见视野时组合视口区域） 在游戏开始时定义游戏窗口或者背景画布的高度，而游戏开始后改变这个值对窗口大小没有明显的影响。如果视口比分配的视野更大或更小，视野将被缩放以适应视口大小，如下图所示：




举例：

view_hview[0] = view_hport[0]; view_wview[0] = view_wport[0];

上述代码将设置视野的宽高与视口的一致。

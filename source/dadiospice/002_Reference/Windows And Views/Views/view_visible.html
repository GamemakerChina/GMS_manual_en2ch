view_visible

是否指定的视野会在屏幕上显示。

语法：

view_visible[0...7]


返回: 布尔值


描述

这个变量可以用来发现一个特定的视野当前是否可见。可以通过设置这个变量的值的true (可见) or false (不可见)，来“打开”或“关闭”一个视野。


举例：

if !view_visible[0]
   {
   view_visible[0] = true;
   }

上述代码检测 view[0] 是否可见，如果不可见，则设置为可见。

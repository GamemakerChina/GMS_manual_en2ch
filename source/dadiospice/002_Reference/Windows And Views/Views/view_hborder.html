view_hborder

视野跟随的水平边界尺寸。

语法：

view_hborder[0...7]


返回： 实数


描述

这个变量设置视野跟随的水平边界框的宽度。. 你可以使用变量 view_object 设置视野跟随指定的实例 (或者在 房间编辑器中设置)，然后设置这个值控制实例需要移动多远或多近视野才会移动。比如，你的视野宽度是 320，设置横向视野边界为160，视野跟随的物体将会一直保持在视野的中心。然而，设置横向视野边界为0，跟随的实例正好在视野边缘时，视野才会移动



注意: 如果没有手动设置视野的x、y位置，视野将不会离开房间的边界。这意味着当视野抵达房间的边缘，会忽略视野的横向或纵向边界，实例将在视野内自由移动。


举例：

view_hborder[0] = view_wview[0] / 2;
view_vborder[0] = view_hview[0] / 2;

上述代码将设置视野跟随边界为视野宽度和高度的一半，有效的迫使跟随的物体保持在视野的中心。

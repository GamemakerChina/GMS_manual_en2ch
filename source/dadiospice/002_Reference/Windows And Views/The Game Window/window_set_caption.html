window_set_caption

设置窗口的标题。

语法：

window_set_caption(caption);


参数 描述
caption The new caption.


返回：N/A


描述

有了这个函数，你可以更改或设置当前所在房间的窗口标题。当游戏不是全屏模式，标题出现在窗口顶部游戏图标的旁边。

注意: 这个函数只适用GameMaker: Studio 专业版。..



举例：

if window_get_caption() != ""
   {
   window_set_caption("");
   }

上述代码检测窗口是否有标题文字，如果有则设置为空，即不显示标题。

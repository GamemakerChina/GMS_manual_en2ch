window_device

返回当前 d3d 设备指针。

语法：

window_device();


返回： 指针


描述

这个函数将返回当前d3d设备的 指针，然后你可以传递给DLL。

注： 这个函数 只对 Windows 和 Mac OSX 有效 (在 Mac OSX 上，它包含一个 NSWindow 类的指针）。


举例：

gfx_pointer = window_device()

上述代码在一个变量中保存d3d设备的指针。


返回: 游戏窗口

window_has_focus

返回游戏窗口是否处于焦点中。

语法：

window_has_focus();


返回: 布尔值


描述

有了这个函数，你可以查询窗口（标签）的状态，如果失去焦点，函数将返回 false ，否则返回 true。大多数情况，简单的使用 os_is_paused 测试焦点是否存在，但是一些特殊的情况（比如Chrome Apps 游戏）那个函数不会触发，这时使用本节所说的这个函数替之。

注意： 这个函数只适用于 HTML5、Windows 8 (JS) 和 Tizen (JS) 模块。


举例：

if !window_has_focus()
   {
   scr_PauseGame();
   }

上述代码将检测游戏窗口是否是焦点，如果返回 false，则调用一个脚本。


返回: 游戏窗口

window_set_min_height

Sets a minimum height for the game window.

语法：

window_set_min_height(height);


参数 描述
height 游戏窗口的最小像素高度。


返回：N/A


描述

这个函数可以给游戏设置一个窗口的最小高度。对于目标平台，如果你在全局游戏设定中允许窗口大小调整，那么玩家将可以调整窗口为任意想要的大小，尽管如此，通过这个函数，你可以限制调整的最小高度为指定值。如果想要回到默认的情况（比如没有最小值限制），只要使用值-1就可以。

注意: 这个函数只适用于 Windows、 Mac OSX 和Linux 平台。


举例：

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

上述代码将为游戏窗口设定最大宽高。

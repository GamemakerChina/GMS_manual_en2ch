url_open_full

通过浏览器，在目标区域打开指定的链接。

语法：

url_open_full(url, target, options);


参数 说明
url 网站链接。
target 在指定的目标区域打开窗口（说明见下文）。
options 标准的浏览器选项（说明见下文）。


返回： N/A


说明

在对应的设备上，程序自动打开浏览器并访问指定的网址。如果是html5平台，则在当前浏览器中打开链接. target 参数用于指定访问链接时，窗口打开方式。
请留意_self'可能会导致 浏览器阻塞，或询问用户是否允许。
（不是所有浏览器都支持所有的自定义选项）。

有效的 targets 如下：

"_self": (GMS默认)在当前的框架中打开被链接文档。
"_blank": 在新窗口中打开被链接文档 (弹出窗口往往是被浏览器限制的，在 这种情况下，你可以使用 clickable_* 函数
"_parent": 在父框架集中打开被链接文档。
"_top": 页面在当前窗口以独立页面显示被链接文档。

有效的 options 如下：

"height=[px]": 窗口的高度, 最小为100.
"width=[px]": 窗口的宽度, 最小为100.
"left=[px]": 相对窗口的坐标 x 值.
"top=[px]": 相对窗口的坐标 y 值.
"location=[boolean]": 地址栏是否显示（默认值为1）.
"menubar=[boolean]": 菜单栏是否显示（默认值为1）.
"resizable=[boolean]": 窗口是否可任意缩放（默认值为1）.
"scrollbars=[boolean]": 滚动条是否显示（默认值为1）.
"status=[boolean]": 状态栏是否显示（默认值为1）.
"titlebar=[boolean]": 标题栏是否显示。浏览器一般会忽略这个属性，除非 调用应用程序为HTML应用程序或值得信赖的对话框 框（默认值为1）.
"toolbar=[boolean]": 工具栏是否显示 （默认显示）.

NOTE: Windows 8 (JS) 平台上, 这个函数也能正常使用，但 target 参数会被忽略，浏览器直接访问指定的ur.


示例：

url_open_full( 'http://yoyogames.com', '_blank', 'resizable=0, height=200, scrollbars=0');

在新的窗口中，访问 YoYo Games 官方网站.这个窗口的高位200像素，无法缩放且没有滚动条

clickable_add_ext

创建一个画布之外的，可点击的DOM元素,并且你可以控制缩放和透明度.

语法：

clickable_add_ext(x, y, tpe, url, target, params, scale, alpha);


参数 说明
x 相对当前浏览器显示窗口的坐标 x 值.
y 相对当前浏览器显示窗口的坐标 y 值.
tpe 精灵图像的材质页映射坐标 (调用 sprite_get_tpe获取).
url URL 地址或者 "GML脚本名+()"(脚本资源名以gmcallback_为前缀).
target 指定在何处打开链接(具体说明看下文).
params 指定打开窗口的属性 (具体说明看下文).
scale 图标的缩放（默认为1）.
alpha 图标的透明度（默认为1）.


返回： 实数


说明

这个函数与 clickable_add 很相似.
调用这个函数，你能够创建一个可自定义属性，可触发点击事件的 图标，并返回这个元素的id，以备后面操作的正常进行.
这个图标就像一个按钮，可以让浏览器创建一个窗口.

如果你想访问URL，那么前缀必须是http:// ，如果你想调用脚本，那么GMS脚本名的前缀必须是gmcallback_
当你调用脚本时，参数 "target" 和 "params" 可以留空字符串("").


示例：

home_but = clickable_add_ext(32, 32,sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 2, 1);
或者：
home_but = clickable_add_ext(32, 32,sprite_get_tpe(spr_MS_Home, 0), "gmcallback_scr_home()" , "", "", 1, 1);

以上代码，在浏览器显示区域坐标(32, 32)处创建一个图标，图标的图像来至于精灵"spr_MS_Home"，并且这个按钮被放大2倍，透明度为1.触发点击事件时，打开一个新的带有指定属性的窗口，并访问指定的 URL 地址,或者调用脚本"gmcallback_scr_home".

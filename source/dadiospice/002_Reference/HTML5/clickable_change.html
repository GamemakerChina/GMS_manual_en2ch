clickable_change

改变图标的精灵。

语法：

clickable_change(index, x, y, tpe)


参数 说明
index 图标的id.
x 相对当前浏览器显示窗口的坐标 x 值.
y 相对当前浏览器显示窗口的坐标 y 值.
tpe 精灵图像的材质页映射坐标 (调用sprite_get_tpe获取).


返回： N/A


说明

使用这个函数，你可以改变已经创建图标（clickable_add）的属性，包括显示精灵以及坐标. 请注意，坐标相对当前浏览器的显示窗口，并非<、i>画布 , 精灵图像的材质页映射坐标 ，调用sprite_get_tpe获取.


示例：

switch (room)
   {
   case rm_Menu: clickable_change(global.Help_Icon, 32, 32, sprite_get_tpe(spr_MS_Help, 1)); break;
   case rm_Game: clickable_change(global.Help_Icon, 200, 32, sprite_get_tpe(spr_MS_Help, 0)); break;
   }

以上代码，根据房间来改变图标的精灵以及坐标，这个图标的 id 存储在变量"global.Help" 中.

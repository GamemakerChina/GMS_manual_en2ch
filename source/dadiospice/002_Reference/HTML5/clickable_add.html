clickable_add

创建一个画布之外的，可点击的DOM元素

语法：

clickable_add(x, y, tpe, url, target, params);


参数 说明
x 相对当前浏览器显示窗口的坐标 x 值.
y 相对当前浏览器显示窗口的坐标 y 值.
tpe 精灵图像的材质页映射坐标 (调用 sprite_get_tpe获取).
url URL 地址或者 "GML脚本名+()"(脚本资源名以gmcallback_为前缀).
target 规定在何处打开链接(具体说明看下文).
params 可用的一些属性 (具体说明看下文).


返回： 实数


描述

调用这个函数，你能够创建一个可自定义属性，可触发点击事件的 图标，并返回这个元素的id，以备后面操作的正常进行 。 这个图标就像一个按钮，对于让浏览器创建一个弹出窗口很有用，并且我们能够很方便的在GMS中操作。

如果，你调用的脚本，那么后面的两个参数可以留空字符串("").

可用的窗口打开方式(targets)：

Target 说明
_blank 在新窗口中打开被链接文档。
_self (GMS默认)在当前的框架中打开被链接文档。
_parent 在父框架集中打开被链接文档。
_top 页面在当前窗口以独立页面显示被链接文档。

可用的窗口参数：

属性 说明
'height=[px]' 窗口的高度, 最小为100.
'width=[px]' 窗口的宽度, 最小为100.
'left=[px]' 相对窗口的坐标 x 值
'top=[px]' 相对窗口的坐标 y 值.
'location=[boolean]' 地址栏是否显示（默认值为1）.
'menubar=[boolean]' 菜单栏是否显示（默认值为1）.
'resizable=[boolean]' 窗口是否可任意缩放（默认值为1）.
'scrollbars=[boolean]' 滚动条是否显示（默认值为1）.
'status=[boolean]' 状态栏是否显示（默认值为1）.
'titlebar=[boolean]' 标题栏是否显示。浏览器一般会忽略这个属性，除非 调用应用程序为HTML应用程序或值得信赖的对话框 框（默认值为1）;
'toolbar=[boolean]' 工具栏是否显示 （默认显示）.

如果你想访问URL，那么前缀必须是http:// ，如果你想调用脚本，那么GMS脚本名的前缀必须是gmcallback_


示例：

home_but = clickable_add(32, 32,sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com","_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
或者：home_but = clickable_add(32, 32,sprite_get_tpe(spr_MS_Home, 0), "gmcallback_scr_home()","", "");

以上代码，在浏览器显示区域坐标(32, 32)处创建一个图标，图标的图像来至于精灵"spr_MS_Home"，触发点击事件时，打开一个新的带有指定属性的窗口，并访问指定的 URL 地址。.

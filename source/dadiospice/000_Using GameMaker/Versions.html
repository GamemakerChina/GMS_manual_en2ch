版本

这部分列出了不同版本 GameMaker: Studio 之间的区别。

有许多不同的 GameMaker: Studio 版本供你选择，每一个版本的规模取决于最终用户的编程经验，无论是 GameMaker 产品或一般产品。所有版本均可编译 Windows 游戏 (如果通过 Steam 客户端下载本软件，还可以发布到 Steam Workshop (创意工坊) )。每个版本除了编译目标不同，还有一些限制，详细说明如下。

如果你想升级你当前版本的 iGameMaker: Studio，你可以访问 YoYo Games 网站，如果你有 Steam 客户端，右键点击 GameMaker: Studio 进入选项选择 "View Downloadable Content (查看下载内容)"。

GameMaker: Studio 免费版

这个版本可以免费下载使用，因为它的目的是允许用户在升级到专业版之前测试该产品的功能。如果你已经使用过以前的 GameMaker 产品，你可以升级到这个版本看看有什么差异，而没使用过 GameMaker 也没关系(或者从未接触过编程)，这个版本是一个好的开始。

下面是标准版 GameMaker: Studio 的限制:

  1. 没有 Configurations (配置)
  2. 没有 Texture Group Control (纹理组控制)
  3. 没有 Extra Target Modules (扩展输出模块)
  4. 没有 DSP (Developer Services Portal 开发者服务门户)
  5. 没有 Source Control (源控制)
  6. 没有 Steam API integration (steam API 集成)

GameMaker: Studio 专业版

专业版 GameMaker: Studio 中所有限制被移除，你可以实验程序的特点:

  1. 为特殊的客户端或输出配置你的产品
  2. 用纹理组控制优化你的游戏和软件
  3. 用 Subversion 为团队工作提供源控制
  4. 开发者服务门户整合内购、广告和分析
  5. 测试开发安卓游戏需要购买安卓模块
  6. Windows 8 应用程序开发 (JavaScript 和 本地包)
  7. 可以额外购买模块(目前有 Mac OSX, HTML5, iOS, Ubuntu, Windows Phone, Android and Tizen)

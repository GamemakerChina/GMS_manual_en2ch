shader_set_uniform_matrix

Set a shader constant to the current transform matrix.

Syntax:

shader_set_uniform_matrix(handle);


Argument Description
handle The handle of the shader constant to set.


Returns: N/A


Description

With this function you can set the value (or values) of a shader constant to the current transform matrix (as set using the D3D Transform Functions). You must previously have gotten the "handle" of the constant using the function shader_get_uniform.


Example:

shader_matrix = shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);

The above code will get the handle of the shader constant "u_vMatrix" then set that constant to the current transform matrix.

Using GameMaker: Studio GameMaker: Studio is an easy to use program for creating your own computer games.		index-index
GameMaker: Studio - Standard only permits you to export games for Windows, and the Professional version also permits you to compile for Windows 8 .	使用 GameMaker: Studio GameMaker:Studio是一个比较容易使用的软件，可以创建你自己的电脑游戏。	index-index
If you have bought GameMaker: Studio through Steam , you will also have an additional target module to export your games to the Steam Workshop .	GameMaker:Studio 可以向windows输出游戏，而专业版可允许向 Windows8平台导出游戏。	index-index
You can also get additional modules which will permit you to export to additional platforms like HTML5, Android, iOS and Windows Phone:	如果你已经通过Steam购买了 GameMaker:Studio ，你将获得额外的模块允许向Steam工场输出游戏。	index-index
The HTML5 module enables you to produce ready-to-run HTML and JavaScript code that you can host or embed on any website The Android module enables you to create and distribute *.apk files If you have an iOS developer account then with the iOS module you can also publish to iPad, iPhone and iPod The Windows Phone module permits you to target those devices for your games (for Windows Surface devices, use the Windows 8 module) With the Tizen module you can target a vast number of "smart" devices With the Mac OSX module you can target the Apple desktop market The Linux module permits you to make desktop games that run on the Ubuntu operating system	你还可以购买其它的附加模块，让你可以制作HTML5、安卓、iOS或者Windows Phone游戏： HTML5模块可以编写即时运行的HTML和 JavaScript代码，然后你可以上传到任何网站 Android模块可以让你创建并发布*.apk文件 如果你有iOS开发者帐户，并且也有iOS模块，你便可以向iPad、iPhone、iPod发现你的游戏 The Windows Phone 模块允许你制作WP设置能运行的游戏，Windows Surface 设备则使用 Windows 8 模块 Tizen 模块 支持很多智能设备 Mac OSX 模块可以向苹果桌面市场发布游戏 The Linux 模块可以让你制作运行在 Ubuntu 操作系统上的桌面游戏。	index-index
These modules are available only to those who have upgraded from the Free version of GameMaker: Studio to the Professional version (more information on the different versions can be found here ).	这些模块只有当你从免费或者普通版本升级到专业版才可以使用（更多不同版本的信息可以在 here 找到）。	index-index

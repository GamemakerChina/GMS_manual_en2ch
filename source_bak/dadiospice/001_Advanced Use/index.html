进阶使用

本节会阑述一些高级的功能，这些功能在 GameMaker: Studio 开发环境中都可以找到。

如果你想把游戏的功能做得更全面， GameMaker: Studio IDE (集成开发环境) 有一些高级的设置将非常重要。这些设置选项包括代码控制、项目配置、高级调试和一些额外的改变、操作游戏资源的工具，尽管如此，你需要根据你使用的GameMaker: Studio 版本选择可用的工具（更多版本的信息可以在这里找到）。接下来的章节里将会解释每个细节，包括 GameMaker: Studio 的高级工具使用、特性和选项。

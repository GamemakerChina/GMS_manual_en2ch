GameMaker: Studio简介。

我们都知道，玩电脑游戏很有趣。但是许多人不知道的是，设计你自己的电脑游戏，让其他人玩实际上更有趣！现在, 制作电脑游戏并不是一件简单的事情，制作一个商业游戏您通常需要花费两年到三年的时间聘请10-50人去制作，很容易达到数百万美元。更不用说，这些人都是经验丰富的程序员、设计师、音响师等。

但是 GameMaker: Studio 的到来改变了这一事实。. 您可以使用 GameMaker: Studio 迅速并轻松地开始创造您自己的电脑游戏，并且不需要学习任何一门编程语言。

GameMaker: Studio 提供了直观且易于拖放 (简称 'DnD' ) 的动作图标，使得您可以很快滴创建您自己的电脑游戏。可以导入或者创建用于物体的图像和声音，然后在按下按钮的时候您就可以立即看到结果。显然，您想尽快地上手，所以 GameMaker: Studio 捆绑了一些教程和演示，您可以直接导入并且运行他们。这些教程包括许多精灵和图形您可以直接使用。但是他们 not 能使用在任何成品里。这里要注意的是 GameMaker: Studio 着重于2D游戏 (并不是说不能制作3D游戏！) 并且很多很棒的游戏都是使用2D精灵技术，即使他们在运行起来像是3D的。

如上图，您需要为物体设计图像 (精灵) 和其他东西，并且通过 DnD 动作图标简单地给它们定义属性和行为，然后将它们和背景以及其他东西放在房间里使得您的游戏看起来像是您想要的那样。 这些房间是游戏运行的地方，你也可以理解为游戏中的 “等级”。如果您掌握了 DnD 界面并且了解它是如何控制物体的基础事件，就可以学习简单的编程语言GML，使得您可以用 GameMaker: Studio控制您的游戏的各个方面。

该帮助文件会告诉你你需要知道关于 GameMaker: Studio 的最重要的事情以及您可以如何使用它创建自己的游戏。请记住，即使拥有一个像 GameMaker: Studio这样的软件，也并没有 “创建一个MMORPG游戏” 按钮使得您毫不费力地创建一个游戏。要知道，制作游戏有很多同样重要的事情：游戏性、画面、声音、用户交互等等，但是，最重要的是您需要有 耐心。而 GameMaker: Studio 是一个强大的工具，它不能够为您提供一个创造一切的按钮，所以从简单的例子开始，您会意识到，不仅制作游戏很好玩，而且学习GameMaker: Studio 也是值得的！

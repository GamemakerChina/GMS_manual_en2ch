view_xview

房间视野的X坐标。

语法：

view_xview[0...7]


返回： 实数


描述

这个变量可以用来获取或设置房间视野的x轴坐标。通过改变这个值可以设置视野至一个新的位置，使用(0,0) 将是房间的左上角，或者参数此变量以确保实例允许绘制在视野以内（查看下面例子中的代码）。这个值也可以是负数以显示房间外面的区域，这种情况下，将会使用到 窗口颜色（window colour） ，这个颜色可以使用window_set_colour进行设置。




举例：

draw_text(view_xview[0] + 32, view_yview[0] + 32, "Score: " + string(score));

上述代码将在相对于视野0（ view[0]）的x、y坐标的一个位置绘制分数。

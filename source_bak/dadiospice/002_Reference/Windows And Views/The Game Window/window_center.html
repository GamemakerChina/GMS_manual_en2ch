window_center

在可视范围或浏览器上将一个区域居中放置。

语法：

window_center();


返回： N/A


描述

对于Windows, Ubuntu (Linux) 或者 MacOS 平台， 你可以使用这个函数将游戏窗口屏幕居中，而对于HTML5目标模块，是浏览器居中。这个函数对移动设备无效。

注： 如果你已经在重置游戏窗口大小（例如切换全屏到窗口），并且希望新的窗口屏幕居中，你应该使用一个alarm[0]（计时器） 来调用这个函数，至少在窗口更改 one step（一步）之后，否则可能会出现问题。


举例：

if !window_get_fullscreen()
   {
   window_center();
   }

在不是全屏模式下的情况下，让游戏窗口屏幕居中放置。

window_set_rectangle

设置窗口矩形的位置和大小。

语法：

window_set_rectangle(x, y, w, h);


参数 描述
x 窗口的新的x坐标。
y 窗口的新的y坐标。
w 窗口的新宽度。
h 窗口的新高度。


返回：N/A


描述

有了这个函数，你可以设置浏览器内（HTML5）或者显示器内（Windows、 Ubuntu （Linux）或者 MacOS）游戏窗口的位置和 大小。更多关于窗口位置和大小的信息，查看 window_set_position 和window_set_size。


举例：

window_set_rectangle(0, 0, display_get_width(), display_get_height());

上述代码将设置游戏窗口覆盖整个显示器区域（可以是浏览器或者屏幕，取决于目标平台）。

instance_count

返回当前在房间内活跃的实例总数。

语法:

instance_count


返回值: 实数


描述

用这个只读变量你能够得到在房间内所有活跃实例的数量。这将包括正在运行代码的实例，但并不包括那些被使用instance_deactivate冻结的实例。要注意这个功能只能给你在每步开始时候的实例数量，所以在一步开始之后任意对房间中实例的改变都将忽略不计。


举例:

if (instance_count < 100)
   {
   var dif = 100 - instance_count;
   while (--dif > 0)
      {
      instance_create(random(room_width), random(room_height), obj_Star);
      }
   }

上述代码将会创建物体"obj_Star"的多个实例直至总的实例数量达到100.

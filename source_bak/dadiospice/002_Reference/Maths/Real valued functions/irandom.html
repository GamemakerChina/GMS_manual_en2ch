irandom

返回一个在[0,n)的随机整实数，如果n本身是一个整数的话则包括n。

语法:

irandom(n);


参数 描述
n 待挑选的随机数范围的上边界。


返回值: 整数


描述

这个有用的函数只返回整数(完整的数)。 因此，举个例子，要得到从0到9的随机数可以使用 irandom(9) 并且它将返回一个从0到9闭区间的随机数……实数也能够被使用但上边界将被排除，所以 irandom(9.7)将只会返回[0,9]上的数。这个函数有一个最大上边界$7fffffffffffffffLL①,所以如果使用非常大的数字时请务必谨慎。

①：$7fffffffffffffffLL表义为long long十六进制数7fffffffffffffff=9223372036854775807D（十进制）

注意:这个函数在每次游戏重新运行时将返回相同的数因为事实上GameMaker: Studio 每次都产生相同的初始随机数种子以使得调试代码成为相当容易的事情。欲避免这个行为请在游戏开始时使用randomize


举例:

if irandom(9) = 1
   {
   score += 100;
   }

这个函数将会产生1/10的几率(因为0被包含) 增加100分。

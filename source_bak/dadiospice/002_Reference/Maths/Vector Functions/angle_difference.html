angle_difference

返回两角的角度差。

语法:

angle_difference(ang1, ang2)


参数 描述
ang1 待使用的第一个角度。
ang2 待使用的第二个角度。


返回值: 实数


描述

这个函数将会返回两个角度之间最小的角度差为一个在-180到180之间的值（其中，一个正角度是逆时针的而一个负角度是顺时针的）。


举例:

var pd = point_direction(x, y, mouse_x, mouse_y);
var dd = angle_difference(image_angle, pd);
image_angle += min(abs(dd), 10) * sign(dd);

上述代码将会获得从实例到鼠标光标的方向角，然后获得其与当前image_angle的角度差，使用这个值来缓慢的朝向鼠标旋转。

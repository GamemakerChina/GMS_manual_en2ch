show_question

显示一个弹出询问

语法:

show_question(str);


参数 描述
str 待显示在弹出询问中的字符串。


返回值: 布尔


描述

这个函数创建一个弹出带有两个按钮为"Yes"和"No"的消息窗口。它将依照用户按下了两个按钮中的哪一个来返回 true 或 false 。

注意: 这个函数仅供调试！


举例:

if score > 500
   {
   if show_question("Continue to next room?")
      {
      room_goto(rm_Level2);
      }
   else game_end();
   }

上述代码将会检测分数并且如果它超过500，它就会询问用户是否还想继续，如果“Yes”按钮被按下，它就会进入另一个房间，但如果"No"按钮被选择，它就会结束游戏。

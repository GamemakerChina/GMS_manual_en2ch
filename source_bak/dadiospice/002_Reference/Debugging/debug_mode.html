debug_mode

告诉你游戏是否以调试模式运行

语法:

debug_mode


返回值: 布尔


描述

当游戏正在被以调试模式运行时，这个只读变量返回 true，而当游戏正在以正常模式运行时则返回false。


Example:

if debug_mode
   {
   ini_open("Cheats.ini");
   }
else
   {
   ini_open("Game.ini");
   }

上述代码依据只读变量debug_mode的值来创建一个不同的ini文件。

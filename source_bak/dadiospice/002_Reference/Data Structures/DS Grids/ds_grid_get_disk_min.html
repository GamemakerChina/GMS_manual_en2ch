ds_grid_get_disk_min

从一个圆形区域中获得所有值的最小值，只有在单元格是数字的时候才生效。

语法:

ds_grid_get_disk_min(index, xm, ym, r);


参数 描述
index 栅格的索引
xm 栅格上圆盘的x位置
ym 栅格上圆盘的y位置
r 栅格上圆盘的半径


返回值: 实数①

①：此处原文为“Real or string”但根据标题所描述，此函数不应当返回字符串。


描述

这个函数可以被用于查找在一个栅格结构中一个圆盘区域以内的所有单元格中的最大值，正如下图所示：


举例:

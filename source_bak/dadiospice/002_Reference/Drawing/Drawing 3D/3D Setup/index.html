3D 初始化

这部分详细介绍初始化3D绘制相关的函数.

以下函数用于设定3D模式相关的基础配置（进入3D模式后，你可以随时切换为2D模式），可以影响3D绘制的效果.

  1. d3d_start
  2. d3d_end
  3. d3d_set_hidden
  4. d3d_set_perspective
  5. d3d_set_depth
  6. d3d_set_culling
  7. d3d_set_zwriteenable

audio_group_name

Returns the name of the audio group as a string.

Syntax:

audio_group_name(groupID);


ArgumentDescription
groupID The index of the audio group to check (as defined in the Global Game Settings Audio Groups Tab)


Returns: String


Description

This function will return a string containing the name of the given audio group for displaying or checking.


Example:

var name= audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

The above code retrieves the name of the given audio group and displays it on the screen.

steam_ugc_download

Find out if the given file exists on the Steam Cloud.

Syntax:

steam_ugc_download(ugc_handle, dest_filename);


ArgumentDescription
ugc_handle The unique handle for the UGC to be downloaded.
dest_filename The file name to save the UGC with.


Returns: Async ID


Description

With this function you can download any given UGC item for your game. The ugc_handle is the unique identifying value for the item (which you can get, for example, using the function steam_ugc_request_item_details()), and the destination filename is the name (and local path within the Steam sandbox) that you wish to give the file when the download is complete. The file type being downloaded will depend on what the item is and how it was uploaded.

When using this function it will trigger an Steam Asynchronous event to report the details of the file requested containing the following key/map values with information about the requested file in the async_load ds_map:

  1. "id" - The async ID returned by the calling function

  2. "result" - The result of the operation (a real value). This will either be the GML constant ugc_result_success or some other real number. So you should check for this constant to ensure that the call was successful, and if otherwise somthing has not worked correctly. The rest of the possible values returned are shown as the result of the Steam "EResult" value and you should see steamclientpublic.h in the SDK headers for all 89 possible values.

  3. "event_type" - This key will hold the value "ugc_download"

  4. "original_filename" - This key holds the original name of the file on the server (a string)

  5. "dest_filename" - This key holds the file name you passed in (a string)

  6. "ugc_handle" - This key holds the ugc_handle value that you passed in to the calling function


Extended Example:

In this example we first call the function and store the async ID value in a variable:

steam_get = steam_ugc_download(steam_handle, "\UGC\Preview_file.dat");

This would then send off a file request to the Steam API, generating an async event which we would deal with as follows:

var event_id = async_load[? "id"];
if event_id == steam_get
   {
   var type = async_load[? "event_type"];
   if type == "ugc_download"
      {
      show_debug_message("Downloading content to " + async_load[? "dest_filename"]);
      }
   }

The above code checks the event type and then outputs a debug message with the destination filename.
